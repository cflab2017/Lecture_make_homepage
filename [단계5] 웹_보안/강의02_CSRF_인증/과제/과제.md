# 과제02 — 로그인 폼 CSRF 방어

## 목표

CSRF 토큰을 hidden input으로 추가하고 서버에서 검증하는 로그인 폼을 구현합니다.

---

## 요구사항

### 1. 로그인 폼 (`/login`)

- CSRF 토큰을 `hidden input`으로 포함
- POST 제출 시 서버에서 CSRF 토큰 검증
- 검증 실패 시 403 반환
- 검증 성공 시 로그인 처리 (메모리 내 사용자 데이터 사용)

### 2. 보호된 라우트 (`/dashboard`)

- 로그인 상태에서만 접근 가능
- 미로그인 시 `/login`으로 리다이렉트
- 로그아웃 버튼 포함 (CSRF 토큰으로 보호)

### 3. 로그아웃 (`POST /logout`)

- CSRF 검증 후 세션 파기

---

## 테스트 계정

```js
const users = [
    { id: 1, username: 'admin', password: 'admin123' },
    { id: 2, username: 'user1', password: 'pass123' },
];
```
