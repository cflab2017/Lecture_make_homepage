# 강의02 과제 — 충돌 해결 연습

## 과제 목표

두 브랜치에서 **같은 파일의 같은 줄**을 수정해 충돌을 발생시키고, 직접 해결하는 연습을 합니다.

## 시나리오

`team-project` 디렉토리에서 두 팀원이 `config.js` 파일을 동시에 수정했습니다.

## 초기 파일 (공통 조상)

```js
// config.js
const config = {
  appName: '팀 프로젝트',
  version: '1.0.0',
  theme: 'default',
  language: 'ko',
  debugMode: false,
};

export default config;
```

## 브랜치 A: feature/settings-update (팀원 A)

다음 내용으로 `config.js`를 수정하세요:
- `theme`를 `'dark'`로 변경
- `apiUrl`를 추가: `apiUrl: 'https://api.example.com'`

## 브랜치 B: feature/locale-update (팀원 B — main에서 직접 수정)

다음 내용으로 `config.js`를 수정하세요:
- `theme`를 `'light'`로 변경 (main에서 작업)
- `language`를 `'en'`으로 변경

## 진행 순서

1. `config.js`로 첫 커밋
2. `feature/settings-update` 브랜치에서 팀원 A 수정
3. `main`으로 돌아와 팀원 B 수정 (같은 `theme` 줄 수정 → 충돌 보장)
4. `git merge feature/settings-update` → 충돌 발생 확인
5. 충돌 해결: `theme`는 `'system'`으로 결정, 나머지 변경사항은 모두 유지

## 최종 목표 파일 (충돌 해결 후)

```js
const config = {
  appName: '팀 프로젝트',
  version: '1.0.0',
  theme: 'system',     // 팀 합의: system (OS 테마 따라감)
  language: 'en',      // feature/locale-update 변경 유지
  debugMode: false,
  apiUrl: 'https://api.example.com',  // feature/settings-update 추가 유지
};
```
