# 과제 01 — 기존 JS 함수에 타입 추가하기

## 목표

아래의 JavaScript 함수 5개를 TypeScript로 변환하여 타입을 완전히 지정하세요.

---

## 변환할 JavaScript 함수

```js
// 1. 두 수의 합
function sum(a, b) {
  return a + b;
}

// 2. 배열의 최댓값
function getMax(arr) {
  return Math.max(...arr);
}

// 3. 문자열 반복
function repeat(str, count) {
  return str.repeat(count);
}

// 4. 짝수만 필터링
function filterEven(numbers) {
  return numbers.filter(n => n % 2 === 0);
}

// 5. 나이로 성인 여부 판단
function isAdult(age) {
  return age >= 18;
}
```

---

## 요구사항

1. 각 함수의 **매개변수 타입**을 지정하세요.
2. 각 함수의 **반환 타입**을 지정하세요.
3. `tsc hello.ts`로 컴파일 오류 없이 통과해야 합니다.
4. 각 함수를 호출하는 테스트 코드를 작성하세요.
