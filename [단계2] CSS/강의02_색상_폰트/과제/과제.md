# 과제 — 브랜드 색상 팔레트

## 목표

CSS 변수(`--custom-property`)를 사용하여 나만의 **브랜드 색상 팔레트** 페이지를 만듭니다.

---

## 요구사항

### 1. CSS 변수 정의

`:root`에 아래 변수를 반드시 정의하세요.

```css
:root {
  --color-main:       ???;  /* 메인 브랜드 색상 */
  --color-sub:        ???;  /* 보조 색상 */
  --color-background: ???;  /* 페이지 배경색 */
  --color-text:       ???;  /* 기본 글자색 */
  --color-accent:     ???;  /* 강조/포인트 색상 */
}
```

색상은 **HEX, RGB, HSL 중 하나**로 표현하세요.

### 2. 팔레트 카드 만들기

5가지 색상을 각각 시각적으로 보여주는 색상 카드를 만드세요.

각 카드에는 다음이 포함되어야 합니다:
- 색상 블록 (정사각형 또는 직사각형)
- 색상 이름 (예: "메인 컬러")
- 색상 값 (예: `#3498db`)

### 3. 폰트 적용

- Google Fonts에서 마음에 드는 폰트를 1개 이상 선택하여 적용하세요.
- `font-family`, `font-size`, `font-weight`, `letter-spacing` 중 3가지 이상 사용하세요.

### 4. 텍스트 활용

- 브랜드 이름(가상의 브랜드도 OK)을 제목으로 표시하세요.
- `text-transform`, `letter-spacing`, `line-height`를 활용하세요.

---

## 파일 구조

```
내폴더/
├── index.html
└── style.css
```

---

## 체크리스트

- [ ] `:root`에 CSS 변수 5개 이상 선언했다
- [ ] `var()` 함수로 색상을 적용했다 (직접 색상값 사용 금지)
- [ ] Google Fonts 폰트를 연결하고 사용했다
- [ ] 색상 카드 5개가 각각 다른 색상을 보여준다
- [ ] 인라인 CSS를 사용하지 않았다
- [ ] 브라우저에서 오류 없이 열린다

---

## 힌트

```html
<!-- 색상 카드 구조 예시 -->
<div class="palette">
  <div class="color-card main-color">
    <div class="color-block"></div>
    <div class="color-info">
      <p class="color-name">메인 컬러</p>
      <p class="color-value">#3498db</p>
    </div>
  </div>
  <!-- ... 나머지 4개 -->
</div>
```

```css
/* 색상 카드 힌트 */
.color-block {
  height: 120px;
  border-radius: 8px 8px 0 0;
  background-color: var(--color-main); /* 변수 활용 */
}
```

> 정답이 궁금하면 `과제정답/` 폴더를 참고하세요.
