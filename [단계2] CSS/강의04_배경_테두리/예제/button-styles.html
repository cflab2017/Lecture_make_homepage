버튼 스타일 쇼케이스

background, border, border-radius, box-shadow, transition 활용

1. 기본 버튼 (Solid)

배경색과 글자색으로 구성된 기본 버튼입니다.

2. 아웃라인 버튼 (Outline)

배경 없이 테두리(border)만 있는 버튼입니다. 호버 시 배경이 채워집니다.

3. 둥근 버튼 (Pill)

border-radius를 크게 설정한 Pill 모양 버튼입니다.

4. 그림자 버튼 (Shadow)

box-shadow로 입체감을 준 버튼입니다. 호버 시 그림자가 변합니다.

5. 다양한 호버 효과

transition과 transform을 활용한 인터랙티브 버튼입니다.
