# 과제 — 내 포트폴리오 반응형 적용

## 목표

`@media` 쿼리를 활용하여 **모바일과 데스크탑 레이아웃을 분리한**  
반응형 포트폴리오 페이지를 만듭니다.

---

## 요구사항

### 1. 필수 구성 요소

- 헤더 (이름 + 네비게이션)
- 히어로 섹션 (자기소개)
- 작업물/기술 카드 섹션 (최소 3개)
- 푸터

### 2. 반응형 조건

| 화면 | 조건 | 레이아웃 |
|------|------|---------|
| 모바일 | ~ 767px | 1열, 네비게이션 세로 배치 |
| 데스크탑 | 768px 이상 | 2~3열, 네비게이션 가로 배치 |

### 3. 필수 @media 쿼리

```css
/* 모바일 기본 스타일 먼저 작성 */
.cards {
  display: grid;
  grid-template-columns: 1fr;   /* 모바일: 1열 */
}

/* 데스크탑 */
@media (min-width: 768px) {
  .cards {
    grid-template-columns: repeat(3, 1fr); /* PC: 3열 */
  }
}
```

### 4. 필수 사용 요소

- [ ] `<meta name="viewport" ...>` 태그
- [ ] `@media (min-width: ...)` 쿼리 최소 2개
- [ ] `clamp()` 또는 `vw` 단위 1개 이상
- [ ] `rem` 단위 활용
- [ ] 모바일/PC 레이아웃이 명확히 구분되어야 함

---

## 체크리스트

- [ ] 뷰포트 메타 태그가 있다
- [ ] 모바일 기준 스타일을 먼저 작성했다 (Mobile-First)
- [ ] `@media (min-width: ...)` 쿼리로 PC 스타일을 추가했다
- [ ] 카드가 모바일에서 1열, PC에서 2열 이상이다
- [ ] 폰트 크기에 `rem` 또는 `clamp()`를 활용했다
- [ ] 인라인 CSS를 사용하지 않았다

---

## 힌트

### Mobile-First 순서

```css
/* 1. 모바일 기본 스타일 */
.nav { flex-direction: column; }
.cards { grid-template-columns: 1fr; }

/* 2. 태블릿 추가 */
@media (min-width: 768px) {
  .nav { flex-direction: row; }
  .cards { grid-template-columns: repeat(2, 1fr); }
}

/* 3. PC 추가 */
@media (min-width: 1024px) {
  .cards { grid-template-columns: repeat(3, 1fr); }
}
```
