# 과제 — 반응형 카드 목록

## 목표

`flex-wrap`을 활용하여 화면 너비에 따라 **자동으로 줄바꿈되는** 카드 목록을 만듭니다.

---

## 요구사항

### 카드 구조

```html
<div class="card-list">
  <div class="card">
    <div class="card-header">아이콘 또는 번호</div>
    <div class="card-body">
      <h3>카드 제목</h3>
      <p>카드 설명 텍스트</p>
    </div>
  </div>
  <!-- 카드 6개 이상 -->
</div>
```

### CSS 조건

| 속성 | 요구사항 |
|------|----------|
| `display: flex` | 카드 컨테이너에 적용 |
| `flex-wrap: wrap` | 화면이 좁으면 줄바꿈 |
| `flex: 1 1 280px` | 카드 최소 너비 280px, 공간 있으면 늘어남 |
| `gap` | 카드 사이 적절한 간격 |
| `justify-content` | `flex-start` 또는 `center` |

### 동작 조건

- **넓은 화면 (900px 이상)**: 3열 배치
- **중간 화면 (600~900px)**: 2열 배치  
- **좁은 화면 (600px 미만)**: 1열 배치

> `flex-wrap: wrap`과 `flex-basis` 조합으로 미디어 쿼리 없이도 구현 가능합니다.

---

## 체크리스트

- [ ] `display: flex`와 `flex-wrap: wrap`을 사용했다
- [ ] `flex: 1 1 280px` (또는 유사 값)으로 카드 크기를 설정했다
- [ ] `gap`으로 카드 사이 간격을 설정했다
- [ ] 카드가 6개 이상 있다
- [ ] 브라우저 창을 좁히면 줄바꿈이 된다
- [ ] 인라인 CSS를 사용하지 않았다

---

## 힌트

```css
.card-list {
  display: flex;
  flex-wrap: wrap;          /* 핵심: 줄바꿈 허용 */
  gap: 20px;
  max-width: 1100px;
  margin: 0 auto;
}

.card {
  flex: 1 1 280px;         /* 최소 280px, 공간 있으면 성장 */
  /* 화면 너비에 따라 자동으로 1~3열 배치 */
}
```
