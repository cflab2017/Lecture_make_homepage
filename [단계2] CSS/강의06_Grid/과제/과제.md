# 과제 — 대시보드 레이아웃

## 목표

`grid-template-areas`를 활용하여 **사이드바 + 메인 영역**으로 구성된  
대시보드 레이아웃을 만듭니다.

---

## 요구사항

### 레이아웃 구조

```
┌─────────────────────────────────────────┐
│              header (헤더)               │ ← 전체 너비
├────────┬────────────────────────────────┤
│        │        main (메인)              │
│sidebar │   ┌────────┬────────┬───────┐  │
│(사이드) │   │ card1  │ card2  │ card3 │  │
│        │   └────────┴────────┴───────┘  │
│        │   ┌──────────────────────────┐ │
│        │   │       chart 영역          │ │
│        │   └──────────────────────────┘ │
├────────┴────────────────────────────────┤
│              footer (푸터)               │ ← 전체 너비
└─────────────────────────────────────────┘
```

### grid-template-areas 사용 (필수)

```css
.dashboard {
  display: grid;
  grid-template-columns: 220px 1fr;    /* 사이드바 고정 + 메인 가변 */
  grid-template-rows: 60px 1fr 50px;   /* 헤더/메인/푸터 */
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  min-height: 100vh;
}
```

### 각 영역 스타일

| 영역 | 스타일 |
|------|--------|
| header | 로고 + 네비게이션 (Flexbox 활용) |
| sidebar | 메뉴 목록 3~5개 |
| main | 통계 카드 3개 + 대형 콘텐츠 영역 |
| footer | 간단한 저작권 표시 |

### 메인 영역 내부 (Grid 다시 사용)

```css
.stats-cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
```

---

## 체크리스트

- [ ] `grid-template-areas`로 레이아웃을 선언했다
- [ ] `grid-area`로 각 요소를 영역에 배치했다
- [ ] 사이드바 너비가 고정(px)이고 메인은 가변(fr)이다
- [ ] 메인 영역 안에 카드 3개를 Grid로 배치했다
- [ ] 인라인 CSS를 사용하지 않았다

---

## 힌트

```css
/* 각 요소에 grid-area 이름 연결 */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }
```
