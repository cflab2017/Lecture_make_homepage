CSS 선택자 실습

다양한 선택자를 눈으로 확인해 보세요.

1. 태그 선택자

이 단락은 태그 선택자(p)로 스타일이 적용됩니다.

모든 p 태그는 동일한 스타일을 공유합니다.

2. 클래스 선택자

이 문장은 .highlight 클래스가 적용되었습니다.

이 문장은 .highlight와 .warning 두 클래스가 모두 적용되었습니다.

이 문장은 클래스 없이 기본 스타일입니다.

3. ID 선택자

이 문장은 #special ID 선택자가 적용되었습니다.

ID는 페이지에서 한 번만 사용해야 합니다.

4. 자손 & 자식 선택자

이 p는 .parent의 직계 자식입니다 (자식 선택자 적용).

이 p는 .parent의 자손이지만 직계 자식이 아닙니다 (자손 선택자 적용).

5. 가상 클래스 선택자

  • 첫 번째 항목 (first-child)
  • 두 번째 항목
  • 세 번째 항목 (nth-child(3))
  • 마지막 항목 (last-child)

마우스를 올려보세요 (:hover 효과)

6. 우선순위 (Specificity)

이 문장에는 태그, 클래스, ID 선택자가 모두 적용됩니다.
