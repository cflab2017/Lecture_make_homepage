시맨틱 HTML이란 무엇인가?

작성일: 2026년 5월 10일 | 작성자: 홍길동 | 카테고리: HTML

시맨틱 HTML은 태그 이름 자체에 의미를 담아 문서의 구조를 표현하는 방법입니다. 단순히 div와 span으로만 구성된 페이지와 달리, 시맨틱 태그를 사용하면 검색 엔진과 스크린리더가 문서를 훨씬 잘 이해할 수 있습니다.

태그: HTML, 시맨틱, SEO

HTML 폼(Form) 완벽 가이드

작성일: 2026년 5월 8일 | 작성자: 홍길동 | 카테고리: HTML

HTML 폼은 사용자로부터 데이터를 입력받는 핵심 요소입니다. 회원가입, 로그인, 검색, 설문조사 등 거의 모든 웹 서비스에서 폼을 사용합니다. 이번 글에서는 input 타입별 사용법부터 fieldset까지 상세히 알아봅니다.

태그: HTML, 폼, input

HTML 테이블로 데이터 표현하기

작성일: 2026년 5월 5일 | 작성자: 김웹 | 카테고리: HTML

테이블은 복잡한 데이터를 행과 열로 정리하여 보여주는 강력한 도구입니다. colspan과 rowspan을 활용하면 합계 행이나 병합 셀을 표현할 수 있습니다. 시간표, 가계부, 성적표 등 다양한 데이터를 표로 나타내 보세요.

태그: HTML, 테이블

HTML 강의 커리큘럼

총 6강으로 구성된 HTML 기초 과정입니다.

  1. 강의 01 - HTML 기초

    DOCTYPE, html/head/body 구조, 기본 태그를 배웁니다.

    수업 시간: 1시간 | 난이도: ⭐

  2. 강의 02 - 텍스트 요소

    strong, em, blockquote, code 등 텍스트를 풍부하게 표현합니다.

    수업 시간: 1시간 | 난이도: ⭐

  3. 강의 03 - 링크와 이미지

    하이퍼링크와 이미지 삽입, 경로 개념을 배웁니다.

    수업 시간: 1시간 30분 | 난이도: ⭐⭐

  4. 강의 04 - 테이블

    표 만들기, colspan/rowspan으로 셀 병합을 배웁니다.

    수업 시간: 1시간 30분 | 난이도: ⭐⭐

  5. 강의 05 - 폼

    다양한 input 타입과 폼 요소를 활용한 입력 양식을 만듭니다.

    수업 시간: 2시간 | 난이도: ⭐⭐⭐
