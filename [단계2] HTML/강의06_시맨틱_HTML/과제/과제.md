# 강의06 과제 - 포트폴리오 페이지 시맨틱 구조 잡기

## 과제 목표

CSS 없이 **시맨틱 HTML 태그만을 사용**하여 나만의 포트폴리오 페이지 구조를 만들어 보세요.  
CSS는 사용하지 않습니다. 오직 HTML 구조와 내용에만 집중합니다.

---

## 요구사항 체크리스트

### 페이지 구조 (필수 시맨틱 태그)
- [ ] `<header>` - 사이트 제목과 메인 내비게이션 포함
- [ ] `<nav>` - 페이지 내 섹션 이동 링크 (앵커 링크 #id 사용)
- [ ] `<main>` - 포트폴리오 핵심 콘텐츠 전체 (페이지에 하나만)
- [ ] `<footer>` - 연락처, 저작권 정보

### main 내부 섹션 구조
- [ ] `<section id="about">` - 나 소개 섹션
- [ ] `<section id="skills">` - 보유 기술 섹션
- [ ] `<section id="projects">` - 프로젝트 섹션 (3개 이상, 각각 `<article>` 사용)
- [ ] `<section id="education">` - 학력/경력 섹션
- [ ] `<aside>` - 포트폴리오 옆 부가 정보 (자기소개 카드, SNS 링크 등)

### article 사용 (프로젝트 섹션)
- [ ] 프로젝트 3개 이상을 각각 `<article>` 로 감싸기
- [ ] 각 article 내에 `<header>` (프로젝트명, 제작 기간) 포함
- [ ] 각 article 내에 `<footer>` (사용 기술, 링크) 포함

### 기타 시맨틱 태그
- [ ] `<time>` 태그로 날짜 표현 (datetime 속성 포함, 최소 2곳)
- [ ] `<figure>` + `<figcaption>` 으로 프로젝트 이미지 표현 (최소 2개)
- [ ] `<address>` 태그로 연락처 정보 표현
- [ ] `<nav>` 내 링크 목록은 `<ul><li><a>` 구조로 작성

### 접근성
- [ ] 모든 `<img>` 에 의미 있는 `alt` 속성 작성
- [ ] `<nav>` 에 주요 페이지 이동 링크 포함

## 포트폴리오 페이지에 포함할 내용

### 1. header 영역
- 이름 또는 닉네임 (h1)
- 한 줄 소개 문구
- 페이지 내 섹션 이동 내비게이션 (소개, 기술, 프로젝트, 학력, 연락처)

### 2. 나 소개 (about) 섹션
- 자기소개 문단 (2~3문장)
- 사진 또는 대표 이미지 (figure + figcaption)

### 3. 보유 기술 (skills) 섹션
- 기술 목록 (ul/li로 정리)
- HTML, CSS, JavaScript 등 알고 있는 기술

### 4. 프로젝트 (projects) 섹션
프로젝트 3개 이상, 각 article에:
- 프로젝트명 (h3)
- 제작 기간 (time 태그 사용)
- 프로젝트 설명 (p)
- 스크린샷 (figure + figcaption)
- 사용 기술, 링크 (footer)

### 5. 학력/경력 (education) 섹션
- 학교명, 전공, 재학기간 (time 태그 사용)
- 수상 내역이나 자격증 (있다면)

### 6. aside 영역
- 간단한 프로필 카드
- SNS/GitHub 링크

### 7. footer 영역
- 이메일, 연락처 (address 태그)
- 저작권 정보 (time 태그로 연도 표현)

## 힌트

<details>
<summary>힌트 1: nav와 앵커 링크로 목차 만들기</summary>

```html
<header>
  <h1>홍길동 포트폴리오</h1>
  <nav>
    <ul>
      <li><a href="#about">소개</a></li>
      <li><a href="#skills">기술</a></li>
      <li><a href="#projects">프로젝트</a></li>
      <li><a href="#education">학력</a></li>
      <li><a href="#contact">연락처</a></li>
    </ul>
  </nav>
</header>

<main>
  <section id="about">...</section>
  <section id="skills">...</section>
  <section id="projects">...</section>
  <section id="education">...</section>
</main>
```
</details>

<details>
<summary>힌트 2: 프로젝트를 article로 표현하기</summary>

```html
<section id="projects">
  <h2>프로젝트</h2>

  <article>
    <header>
      <h3>나의 첫 번째 웹 페이지</h3>
      <p>
        제작 기간:
        <time datetime="2026-03">2026년 3월</time> ~
        <time datetime="2026-04">2026년 4월</time>
      </p>
    </header>

    <figure>
      <img src="https://picsum.photos/seed/project1/500/300" alt="첫 번째 프로젝트 스크린샷">
      <figcaption>나의 첫 자기소개 페이지 스크린샷</figcaption>
    </figure>

    <p>HTML과 CSS를 처음 배우며 만든 자기소개 페이지입니다. 기본 구조와 스타일링을 연습했습니다.</p>

    <footer>
      <p>사용 기술: HTML, CSS</p>
      <p>
        <a href="https://github.com" target="_blank" rel="noopener noreferrer">GitHub 보기</a> |
        <a href="#" target="_blank" rel="noopener noreferrer">라이브 데모</a>
      </p>
    </footer>
  </article>

</section>
```
</details>

<details>
<summary>힌트 3: aside로 프로필 카드 만들기</summary>

```html
<aside>
  <section>
    <h2>프로필</h2>
    <figure>
      <img src="https://picsum.photos/seed/me/150/150" alt="홍길동 프로필 사진" width="150">
      <figcaption>홍길동</figcaption>
    </figure>
    <p>웹 개발을 공부하는 학생 개발자</p>
    <address>
      <a href="mailto:hong@example.com">hong@example.com</a>
    </address>
  </section>

  <section>
    <h2>소셜 링크</h2>
    <nav>
      <ul>
        <li><a href="https://github.com" target="_blank" rel="noopener noreferrer">GitHub</a></li>
        <li><a href="https://linkedin.com" target="_blank" rel="noopener noreferrer">LinkedIn</a></li>
      </ul>
    </nav>
  </section>
</aside>
```
</details>

<details>
<summary>힌트 4: time 태그 올바르게 사용하기</summary>

```html
<!-- 연도만 -->
<time datetime="2026">2026년</time>

<!-- 연월 -->
<time datetime="2026-03">2026년 3월</time>

<!-- 연월일 -->
<time datetime="2026-05-10">2026년 5월 10일</time>

<!-- 기간 표현 -->
<p>재학 기간: <time datetime="2022-03">2022년 3월</time> ~ <time datetime="2026-02">2026년 2월</time></p>
```
</details>
