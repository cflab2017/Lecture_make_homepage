# 강의01 과제 - 우리 가족 소개 페이지 만들기

## 과제 목표

HTML 기초 태그를 활용하여 **우리 가족을 소개하는 웹 페이지**를 만들어 보세요.

---

## 요구사항 체크리스트

아래 항목을 모두 포함하여 페이지를 완성하세요.

### 필수 구조
- [ ] `<!DOCTYPE html>` 선언 포함
- [ ] `<html lang="ko">` 최상위 태그 사용
- [ ] `<head>` 영역에 `<meta charset="UTF-8">` 포함
- [ ] `<head>` 영역에 `<title>` 포함 (예: "우리 가족 소개")
- [ ] `<body>` 안에 모든 내용 작성

### 제목 태그 (h1~h6)
- [ ] `<h1>` 태그로 페이지 전체 제목 작성 (예: "우리 가족을 소개합니다!")
- [ ] `<h2>` 태그로 각 가족 구성원의 소제목 작성 (최소 3명)

### 단락 태그 (p)
- [ ] 각 가족 구성원에 대한 소개 문장을 `<p>` 태그로 작성 (1인당 최소 1개)
- [ ] 가족 전체에 대한 소개 문단 1개 포함

### 목록 태그 (ul, ol, li)
- [ ] `<ul>`을 사용하여 가족 구성원 목록 작성 (이름, 나이, 관계 등)
- [ ] `<ol>`을 사용하여 가족과 함께 하고 싶은 것 목록 작성 (3가지 이상, 순서 있게)

### 추가 사항
- [ ] `<div>` 또는 `<span>`을 최소 1회 이상 사용
- [ ] HTML 주석(`<!-- -->`)을 최소 2곳 이상 작성

---

## 예시 구조 (참고용)

```
우리 가족을 소개합니다!  ← h1

우리 가족은 총 4명입니다...  ← p

가족 구성원 목록  ← h2
- 아버지 (홍판서, 55세)   ← ul > li
- 어머니 (춘향모, 50세)
- 나 (홍길동, 25세)
- 동생 (홍길순, 20세)

아버지  ← h2
아버지는 회사원이시고...  ← p

어머니  ← h2
어머니는 선생님이시고...  ← p

우리 가족이 함께 하고 싶은 것  ← h2
1. 제주도 여행       ← ol > li
2. 캠핑 가기
3. 외식하기
```

## 힌트

<details>
<summary>힌트 1: 기본 구조 템플릿</summary>

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>우리 가족 소개</title>
</head>
<body>
  <!-- 여기에 내용을 작성하세요 -->
</body>
</html>
```
</details>

<details>
<summary>힌트 2: ul과 ol의 차이</summary>

```html
<!-- ul: 순서가 없는 목록 (•, -, * 등으로 표시) -->
<ul>
  <li>아버지</li>
  <li>어머니</li>
  <li>나</li>
</ul>

<!-- ol: 순서가 있는 목록 (1, 2, 3... 으로 표시) -->
<ol>
  <li>첫 번째로 하고 싶은 것</li>
  <li>두 번째로 하고 싶은 것</li>
  <li>세 번째로 하고 싶은 것</li>
</ol>
```
</details>

<details>
<summary>힌트 3: 중첩 목록 사용하기</summary>

가족 구성원의 취미도 같이 소개하려면 중첩 목록을 사용할 수 있어요:

```html
<ul>
  <li>아버지
    <ul>
      <li>취미: 낚시</li>
      <li>직업: 회사원</li>
    </ul>
  </li>
  <li>어머니
    <ul>
      <li>취미: 독서</li>
      <li>직업: 선생님</li>
    </ul>
  </li>
</ul>
```
</details>
