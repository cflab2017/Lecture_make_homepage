# 강의05 과제 - 설문조사 페이지 만들기

## 과제 목표

이번 강의에서 배운 다양한 폼 요소를 활용하여  
**주제를 자유롭게 선택하여 5문항 이상의 설문조사 페이지**를 만들어 보세요.

---

## 설문 주제 예시 (자유롭게 선택)

- 동아리/모임 신청서
- 독서 취향 조사
- 급식 메뉴 만족도 조사
- 웹 개발 학습 경험 조사
- 여행 계획 설문
- 수강 신청 양식

---

## 요구사항 체크리스트

### form 태그
- [ ] `<form>` 태그 사용 (`action`과 `method` 속성 포함)
- [ ] `method="post"` 사용 (개인정보가 있으므로)

### 필수 input 타입 (다음 중 최소 5가지 이상 사용)
- [ ] `type="text"` - 단답형 텍스트 입력
- [ ] `type="email"` - 이메일 입력 (자동 형식 검증)
- [ ] `type="number"` - 숫자 입력
- [ ] `type="radio"` - 단일 선택
- [ ] `type="checkbox"` - 다중 선택
- [ ] `type="date"` - 날짜 선택
- [ ] `type="range"` - 범위 슬라이더
- [ ] `type="file"` - 파일 업로드

### select와 textarea
- [ ] `<select>` 드롭다운 메뉴 1개 이상 사용
- [ ] `<textarea>` 여러 줄 텍스트 입력 1개 이상 사용

### label과 구조
- [ ] 모든 입력 요소에 `<label>` 연결 (`for`-`id` 또는 감싸기 방식)
- [ ] `<fieldset>`과 `<legend>`으로 관련 질문 그룹화 (2그룹 이상)

### 검증 속성
- [ ] `required` 속성 최소 3곳 이상 사용
- [ ] `placeholder` 속성 최소 2곳 이상 사용
- [ ] `min`, `max` 또는 `minlength`, `maxlength` 최소 1곳 이상

### 버튼
- [ ] `type="submit"` 제출 버튼
- [ ] `type="reset"` 초기화 버튼

---

## 설문 문항 구성 가이드 (최소 5문항)

각 문항은 아래 중 하나의 형식을 사용하세요:

| 문항 유형 | 적합한 입력 요소 |
|-----------|----------------|
| 이름/이메일 등 단순 정보 | `input type="text"`, `type="email"` |
| "다음 중 하나를 선택하세요" | `input type="radio"` |
| "해당하는 것을 모두 선택하세요" | `input type="checkbox"` |
| "목록에서 선택하세요" | `select` |
| "만족도를 1~10으로 표시하세요" | `input type="range"` 또는 `type="number"` |
| "자세히 설명해 주세요" | `textarea` |
| "날짜를 선택하세요" | `input type="date"` |
| "파일을 첨부하세요" | `input type="file"` |

## 힌트

<details>
<summary>힌트 1: 라디오 버튼과 체크박스 구분</summary>

```html
<!-- radio: 그룹에서 하나만 선택 (name이 같아야 그룹) -->
<p>당신의 성별은?</p>
<input type="radio" id="male" name="gender" value="male">
<label for="male">남성</label>
<input type="radio" id="female" name="gender" value="female">
<label for="female">여성</label>

<!-- checkbox: 여러 개 선택 가능 -->
<p>좋아하는 음식을 모두 선택하세요:</p>
<input type="checkbox" id="food1" name="food" value="pizza">
<label for="food1">피자</label>
<input type="checkbox" id="food2" name="food" value="sushi">
<label for="food2">초밥</label>
<input type="checkbox" id="food3" name="food" value="chicken">
<label for="food3">치킨</label>
```
</details>

<details>
<summary>힌트 2: fieldset으로 질문 그룹화</summary>

```html
<form action="/survey" method="post">

  <fieldset>
    <legend>기본 정보</legend>
    <!-- 이름, 이메일, 나이 등 -->
  </fieldset>

  <fieldset>
    <legend>설문 문항</legend>
    <!-- 주요 설문 질문들 -->
  </fieldset>

  <fieldset>
    <legend>추가 의견</legend>
    <!-- textarea 등 -->
  </fieldset>

  <button type="submit">제출</button>
  <button type="reset">초기화</button>
</form>
```
</details>

<details>
<summary>힌트 3: select로 드롭다운 만들기</summary>

```html
<label for="satisfaction">전체 만족도:</label>
<select id="satisfaction" name="satisfaction" required>
  <option value="">-- 선택하세요 --</option>
  <option value="5">매우 만족</option>
  <option value="4">만족</option>
  <option value="3">보통</option>
  <option value="2">불만족</option>
  <option value="1">매우 불만족</option>
</select>
```
</details>

<details>
<summary>힌트 4: range 슬라이더 사용하기</summary>

```html
<label for="recommend">추천 지수 (1~10): <output id="recommend-value">5</output></label>
<input
  type="range"
  id="recommend"
  name="recommend_score"
  min="1"
  max="10"
  step="1"
  value="5"
>
<!-- 슬라이더 값을 실시간으로 표시하려면 JavaScript가 필요합니다 -->
<!-- 지금은 숫자가 자동으로 바뀌지 않아도 괜찮습니다 -->
```
</details>
