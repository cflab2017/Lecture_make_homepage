# 강의04 과제 - 나의 월간 지출 내역표 만들기

## 과제 목표

HTML 테이블 태그를 사용하여 **한 달간의 개인 지출 내역**을 표로 만들어 보세요.

---

## 요구사항 체크리스트

### 기본 테이블 구조
- [ ] `<table>` 태그로 테이블 전체 감싸기
- [ ] `<caption>` 태그로 테이블 제목 작성 (예: "2026년 5월 지출 내역")
- [ ] `<thead>`, `<tbody>`, `<tfoot>` 3개 영역으로 테이블 구분
- [ ] `<tr>`, `<th>`, `<td>` 적절히 사용

### 머리글 (thead)
- [ ] 열 제목: 날짜, 카테고리, 항목, 금액 (4개 이상)
- [ ] 모든 `<th>` 에 `scope="col"` 속성 추가

### 본문 (tbody)
- [ ] 최소 **10개 이상의 지출 항목** 입력
- [ ] **카테고리별**로 최소 3개 카테고리 (예: 식비, 교통비, 여가, 통신비 등)
- [ ] 행 제목(`<th scope="row">`)으로 카테고리를 표시하고 `rowspan` 사용

### 셀 병합 (필수)
- [ ] `colspan` 최소 1회 사용 (예: 합계 행에서 "날짜"+"카테고리"+"항목" 3칸 병합)
- [ ] `rowspan` 최소 1회 사용 (예: 같은 카테고리의 여러 항목을 하나로 묶기)

### 바닥글 (tfoot)
- [ ] 카테고리별 소계 또는 전체 합계 행 포함
- [ ] 합계 행에서 적절한 `colspan` 사용

---

## 표 구성 예시

```
| 날짜  | 카테고리 | 항목         | 금액        |
|-------|----------|-------------|------------|
| 5/1   |          | 편의점       | 3,500원    |
| 5/3   |  식비    | 점심 식사    | 8,000원    |
| 5/8   |          | 치킨 배달    | 20,000원   |
| 5/5   | 교통비   | 지하철 충전  | 10,000원   |
| 5/10  |          | 버스 카드    | 5,000원    |
|       |          | [식비 소계]  | 31,500원   |
|       |          | [교통 소계]  | 15,000원   |
|       | [전체 합계]              | 46,500원   |
```

`rowspan`을 사용하면 같은 카테고리의 여러 행에서 카테고리 셀 하나가 여러 행을 차지하게 됩니다.

---

## 힌트

<details>
<summary>힌트 1: rowspan으로 카테고리 묶기</summary>

같은 카테고리에 속하는 항목이 3개라면 `rowspan="3"` 을 사용합니다:

```html
<tbody>
  <tr>
    <td>5/1</td>
    <td rowspan="3">식비</td>  <!-- 식비 셀이 3행을 차지 -->
    <td>편의점</td>
    <td>3,500원</td>
  </tr>
  <tr>
    <td>5/3</td>
    <!-- 식비 셀은 이미 위에서 rowspan으로 차지했으므로 여기엔 없음 -->
    <td>점심 식사</td>
    <td>8,000원</td>
  </tr>
  <tr>
    <td>5/8</td>
    <!-- 식비 셀 생략 -->
    <td>치킨 배달</td>
    <td>20,000원</td>
  </tr>
</tbody>
```
</details>

<details>
<summary>힌트 2: tfoot에서 colspan으로 합계 표시</summary>

```html
<tfoot>
  <tr>
    <!-- 날짜, 카테고리, 항목 3칸을 합쳐서 "5월 총 지출" 표시 -->
    <td colspan="3">5월 총 지출</td>
    <td>XXX,XXX원</td>
  </tr>
</tfoot>
```
</details>

<details>
<summary>힌트 3: 카테고리별 소계 행 만들기</summary>

소계 행은 tbody 안에서 카테고리가 끝날 때마다 추가합니다:

```html
<tbody>
  <!-- 식비 항목들... -->
  <tr>
    <!-- 날짜 + 카테고리 두 칸 병합으로 "식비 소계" 표시 -->
    <td colspan="2">식비 소계</td>
    <td>항목 수: 3개</td>
    <td>31,500원</td>
  </tr>
</tbody>
```
</details>

<details>
<summary>힌트 4: scope 속성 사용하기</summary>

```html
<!-- 열 제목 -->
<thead>
  <tr>
    <th scope="col">날짜</th>
    <th scope="col">카테고리</th>
    <th scope="col">항목</th>
    <th scope="col">금액</th>
  </tr>
</thead>

<!-- 행 제목 (카테고리를 th로 표현할 때) -->
<tbody>
  <tr>
    <td>5/1</td>
    <th scope="row" rowspan="3">식비</th>  <!-- 이 th가 3행의 제목 -->
    <td>편의점</td>
    <td>3,500원</td>
  </tr>
</tbody>
```
</details>
