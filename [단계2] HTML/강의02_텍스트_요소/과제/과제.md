# 강의02 과제 - 좋아하는 책 또는 영화 리뷰 페이지 만들기

## 과제 목표

이번 강의에서 배운 다양한 텍스트 태그를 활용하여  
**좋아하는 책 또는 영화의 리뷰 페이지**를 만들어 보세요.

---

## 요구사항 체크리스트

### 기본 구조
- [ ] HTML5 기본 구조 (DOCTYPE, html, head, body) 작성
- [ ] `<title>` 에 리뷰 페이지 제목 작성 (예: "내가 좋아하는 책 리뷰")
- [ ] `<h1>` 으로 책/영화 제목 작성

### 강조 태그 사용
- [ ] `<strong>` 태그를 최소 2회 이상 사용 (중요한 내용 강조)
- [ ] `<em>` 태그를 최소 2회 이상 사용 (강세 표현)
- [ ] `<mark>` 태그로 핵심 내용 하이라이트

### 인용 태그 사용
- [ ] `<blockquote>` 로 책/영화의 명대사 또는 명문장 인용
- [ ] `<cite>` 로 책/영화 제목 또는 저자/감독 표시
- [ ] `<q>` 로 짧은 인용문 1개 이상 사용

### 코드/서식 태그 사용
- [ ] `<br>` 줄바꿈을 적절히 사용 (주소, 시 등)
- [ ] `<hr>` 으로 섹션 구분 (최소 2개)

### 기타 태그
- [ ] `<del>` 또는 `<ins>` 태그 사용 (예: 수정된 평점, 변경된 정보)
- [ ] `<sup>` 또는 `<sub>` 태그 사용 (각주 번호, 화학식 등)

---

## 리뷰 페이지에 포함할 내용

다음 항목들을 모두 포함하여 페이지를 작성하세요:

1. **책/영화 기본 정보**
   - 제목, 저자(감독), 출판연도(개봉연도), 장르

2. **줄거리 요약** (p 태그로 작성)
   - 스포일러 없이 2~3문단으로 요약

3. **인상 깊은 명대사 또는 구절** (blockquote 사용)

4. **나의 리뷰**
   - 좋았던 점, 아쉬웠던 점
   - 별점 (예: ★★★★☆ - 5점 만점에 4점)

5. **추천 대상**
   - 이 책/영화를 추천하고 싶은 사람

---

## 힌트

<details>
<summary>힌트 1: 별점 표현하기</summary>

```html
<!-- sup 태그를 활용한 각주 스타일 별점 -->
<p>나의 평점: ★★★★☆ (5점 만점에 4점<sup>*</sup>)</p>
<p><sup>*</sup> 개인적인 취향에 따라 다를 수 있습니다.</p>

<!-- del과 ins로 평점 변경 표현 -->
<p>최초 평점: <del>3점</del> → <ins>4점</ins> (재감상 후 평점 수정)</p>
```
</details>

<details>
<summary>힌트 2: blockquote 올바르게 사용하기</summary>

```html
<blockquote>
  <p>"이 책에서 가장 인상 깊었던 문장을 여기에 적어주세요."</p>
  <footer>
    — <cite>책 제목</cite>, 저자명, p. 123
  </footer>
</blockquote>
```
</details>

<details>
<summary>힌트 3: 줄바꿈 올바르게 사용하기</summary>

```html
<!-- br은 단순 줄바꿈. 시나 주소 등에 적합 -->
<p>
  출판사: 민음사<br>
  출판 연도: 2020년<br>
  페이지 수: 384쪽<br>
  장르: 소설, SF
</p>

<!-- 새로운 단락은 p 태그로 분리하는 것이 올바른 방법 -->
<p>첫 번째 단락</p>
<p>두 번째 단락</p>
```
</details>

<details>
<summary>힌트 4: 리뷰 페이지 구조 예시</summary>

```html
<h1>[책 제목]</h1>

<p>저자: ... | 출판연도: ... | 장르: ...</p>

<hr>

<h2>줄거리</h2>
<p>내용...</p>

<hr>

<h2>인상 깊은 구절</h2>
<blockquote>
  <p>"명언..."</p>
  <footer>— <cite>책 제목</cite></footer>
</blockquote>

<hr>

<h2>나의 리뷰</h2>
<h3>좋았던 점</h3>
<p>...</p>
<h3>아쉬웠던 점</h3>
<p>...</p>
<p>평점: ★★★★☆</p>

<hr>

<h2>이런 분께 추천합니다</h2>
<p>...</p>
```
</details>
