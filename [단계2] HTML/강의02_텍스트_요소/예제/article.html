웹 개발 시작하기: HTML이란 무엇인가?

작성일: 2026년 5월 10일 | 작성자: 홍길동

들어가며

웹 개발을 처음 시작하는 분들이 가장 먼저 접하는 것이 바로 HTML입니다. HTML은 웹 페이지의 구조와 내용을 정의하는 언어로, 모든 웹 개발의 출발점이 됩니다.

이 글에서는 HTML이 무엇인지, 왜 중요한지, 그리고 어떻게 시작할 수 있는지에 대해 상세하게 설명해 드리겠습니다.

HTML이란?

HTML은 HyperText Markup Language의 약자입니다. 이름에서 알 수 있듯이, HTML은 프로그래밍 언어가 아니라 마크업 언어입니다. 즉, 논리적인 계산이나 조건 처리를 하는 것이 아니라, 문서의 구조와 의미를 태그(tag)로 표시(mark up)하는 언어입니다.

HTML의 역사

HTML은 1991년 팀 버너스리(Tim Berners-Lee)에 의해 처음 만들어졌습니다. 현재 우리가 사용하는 버전은 HTML5로, 2014년 W3C에 의해 표준화되었습니다.

왜 HTML을 배워야 할까?

웹 개발자가 되고 싶다면 HTML은 절대적으로 필수입니다. CSS와 JavaScript를 아무리 잘 알아도, HTML 없이는 웹 페이지를 만들 수 없습니다.

"HTML은 웹을 이루는 가장 기초적인 구성 요소입니다. HTML은 웹 콘텐츠의 의미와 구조를 정의할 때 사용합니다."

— MDN Web Docs

많은 개발자들이 HTML은 하루만에 배울 수 있다고 말하지만, 올바르게 사용하는 데는 꾸준한 연습이 필요합니다.

HTML 코드 살펴보기

가장 기본적인 HTML 문서는 아래와 같이 생겼습니다. <!DOCTYPE html>으로 시작해서 <html>, <head>, <body> 태그로 구성됩니다. 


<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <h1>안녕하세요!</h1>
    <p>첫 번째 HTML 문서입니다.</p>
</body>
</html>

이처럼 HTML 태그는 항상 <태그명>으로 시작해서 </태그명>으로 끝납니다.

수학과 화학에서의 HTML 활용

HTML의 위첨자(<sup>)와 아래첨자(<sub>) 태그를 사용하면 수학 공식이나 화학식도 표현할 수 있습니다.

수학 공식

피타고라스 정리: a2 + b2 = c2

아인슈타인 방정식: E = mc2

화학식

물: H2O

이산화탄소: CO2

포도당: C6H12O6
