# 강의 02 - 텍스트 요소

> 글을 더 풍부하고 의미 있게 표현하는 다양한 텍스트 태그들을 배웁니다.

---

## 1. 강조 태그

### strong - 강한 강조 (의미적 중요성)

```html
<p>이 약은 <strong>하루 3번</strong> 복용해야 합니다.</p>
<p><strong>주의:</strong> 이 내용은 매우 중요합니다.</p>
```

- 브라우저에서 **굵게(bold)** 표시됩니다.
- 단순히 굵게 보이게 하는 것이 아니라, **내용이 중요하다는 의미**를 담고 있습니다.
- 스크린리더(시각장애인용 도구)가 강조하여 읽어줍니다.

### em - 강조 (억양, 강세)

```html
<p>나는 <em>정말</em> 배가 고프다.</p>
<p>그건 <em>내가</em> 한 게 아니야.</p>
```

- 브라우저에서 *기울임(italic)* 으로 표시됩니다.
- 특정 단어에 **말하는 억양의 강세**를 주는 의미입니다.

### b와 i - 시각적 표현 (의미 없음)

```html
<p><b>굵은 글씨</b>이지만 특별히 중요한 의미는 없습니다.</p>
<p><i>기울어진 글씨</i>이지만 강조의 의미는 없습니다.</p>
```

| 태그 | 시각 효과 | 의미적 중요성 |
|------|-----------|--------------|
| `<strong>` | 굵게 | 있음 (중요한 내용) |
| `<b>` | 굵게 | 없음 (단순 스타일) |
| `<em>` | 기울임 | 있음 (강세/강조) |
| `<i>` | 기울임 | 없음 (기술 용어, 외국어 등) |

> **권장**: 가능하면 `<b>`, `<i>` 보다 `<strong>`, `<em>`을 사용하세요.  
> 의미 있는 HTML 작성이 SEO와 접근성에 더 유리합니다.

---

## 2. 인용 태그

### blockquote - 블록 인용문

다른 출처에서 가져온 긴 인용문을 나타냅니다. 블록 레벨 요소입니다.

```html
<blockquote cite="https://www.example.com">
  <p>
    "Education is not the filling of a pail, but the lighting of a fire."
  </p>
</blockquote>
```

- `cite` 속성: 인용 출처의 URL을 명시 (화면에는 보이지 않음)
- 브라우저에서 들여쓰기로 표시됩니다.

### cite - 작품 제목 / 출처

책, 영화, 노래 등 창작물의 제목을 나타냅니다.

```html
<p>
  <cite>해리포터와 마법사의 돌</cite>은 J.K. 롤링이 쓴 소설입니다.
</p>

<blockquote>
  <p>"그래도 지구는 돈다."</p>
  <footer>— <cite>갈릴레오 갈릴레이</cite></footer>
</blockquote>
```

### q - 인라인 인용문

짧은 인용문을 문장 안에서 사용할 때 씁니다.

```html
<p>선생님은 <q>열심히 하면 반드시 된다</q>고 말씀하셨습니다.</p>
```

- 브라우저가 자동으로 인용 부호(`" "`)를 붙여줍니다.

## 3. 코드 관련 태그

### code - 인라인 코드

```html
<p>변수를 선언할 때는 <code>let</code> 또는 <code>const</code>를 사용합니다.</p>
<p>HTML에서 단락은 <code>&lt;p&gt;</code> 태그를 사용합니다.</p>
```

- 주로 태그 이름, 함수명, 변수명 등 코드 한 줄을 표현할 때 사용합니다.
- 브라우저에서 고정폭 글꼴(monospace)로 표시됩니다.

### pre - 서식이 있는 텍스트 블록

```html
<pre>
  function greet(name) {
    console.log("안녕하세요, " + name + "!");
  }

  greet("홍길동");
</pre>
```

- 입력한 **공백, 줄바꿈이 그대로 유지**됩니다.
- 보통 `<code>`와 함께 사용하여 코드 블록을 표현합니다.

```html
<pre><code>
  def hello():
      print("Hello, World!")

  hello()
</code></pre>
```

## 4. 줄바꿈과 구분선

### br - 줄바꿈

```html
<p>
  서울특별시 강남구<br>
  테헤란로 123<br>
  4층 401호
</p>
```

- 빈 태그로 닫는 태그가 없습니다.
- 새 단락이 아닌 단순 줄바꿈이 필요할 때 사용합니다.
- **남용 주의**: 단락 사이 여백을 위해 `<br>`을 여러 번 쓰는 것은 좋지 않습니다. CSS를 사용하세요.

### hr - 수평선 (주제 구분)

```html
<p>첫 번째 섹션의 내용입니다.</p>

<hr>

<p>두 번째 섹션의 내용입니다. 위의 hr 태그가 두 섹션을 시각적으로 구분합니다.</p>
```

- 주제의 전환이나 단락의 구분을 나타냅니다.
- 빈 태그입니다.

---

## 5. 위첨자와 아래첨자

### sup - 위첨자 (superscript)

```html
<p>물의 화학식은 H<sub>2</sub>O이고, 이산화탄소는 CO<sub>2</sub>입니다.</p>
<p>피타고라스 정리: a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup></p>
<p>각주 내용<sup>1</sup>을 표시할 때도 사용합니다.</p>
```

### sub - 아래첨자 (subscript)

```html
<p>물의 분자식: H<sub>2</sub>O</p>
<p>포도당: C<sub>6</sub>H<sub>12</sub>O<sub>6</sub></p>
```

---

## 6. 기타 유용한 태그

### mark - 형광펜 효과

```html
<p>시험 범위는 <mark>3장부터 5장까지</mark>입니다.</p>
```

### del - 삭제된 텍스트

```html
<p>가격: <del>50,000원</del> → 35,000원 (특별 할인)</p>
```

### ins - 삽입된 텍스트

```html
<p>회의 날짜: <del>3월 5일</del> <ins>3월 7일</ins>로 변경되었습니다.</p>
```

### abbr - 약어

```html
<p><abbr title="HyperText Markup Language">HTML</abbr>은 웹의 기본 언어입니다.</p>
```

- `title` 속성에 약어의 전체 표현을 적습니다.
- 마우스를 올리면 툴팁으로 표시됩니다.

| `<code>` | 인라인 코드 | 인라인 |
| `<pre>` | 서식 보존 텍스트 블록 | 블록 |
| `<br>` | 줄바꿈 | 인라인 |
