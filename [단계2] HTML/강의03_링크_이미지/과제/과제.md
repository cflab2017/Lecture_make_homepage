# 강의03 과제 - 내 즐겨찾기 링크 모음 페이지 만들기

## 과제 목표

링크와 이미지 태그를 활용하여 **자신이 자주 방문하는 사이트 5개**를 소개하는  
**즐겨찾기 링크 모음 페이지**를 만들어 보세요.

---

## 요구사항 체크리스트

### 링크 (a 태그)
- [ ] 소개할 사이트 **최소 5개** 링크 포함
- [ ] 모든 외부 링크에 `target="_blank"` 사용 (새 탭에서 열기)
- [ ] 모든 외부 링크에 `rel="noopener noreferrer"` 사용 (보안)
- [ ] `mailto:` 형식의 이메일 링크 1개 포함
- [ ] 페이지 내 이동 앵커 링크(`href="#id"`) 최소 2개 사용
- [ ] 페이지 맨 위로 돌아가는 링크 1개 이상 포함

### 이미지 (img 태그)
- [ ] 각 사이트 소개에 관련 이미지 포함 (온라인 URL 사용 가능)
- [ ] 모든 `<img>` 태그에 `alt` 속성 작성 (의미 있는 설명)
- [ ] `width` 속성으로 이미지 크기 지정
- [ ] 이미지를 클릭하면 해당 사이트로 이동하는 **이미지 링크** 최소 1개

### figure/figcaption
- [ ] `<figure>` 와 `<figcaption>` 을 사용하여 이미지 + 설명 묶기 (최소 3개)

### 구조
- [ ] 카테고리별로 섹션 나누기 (h2 사용, 예: 뉴스, 엔터테인먼트, 개발 등)
- [ ] 각 사이트에 대한 짧은 설명 포함 (p 태그 사용)
- [ ] `<hr>` 로 카테고리 구분

---

## 사이트 소개에 포함할 내용

각 사이트마다 다음 내용을 포함하세요:

1. **사이트 이름** (h3 태그 사용)
2. **사이트 주소** (a 태그로 링크)
3. **사이트 소개** (p 태그로 2~3문장)
4. **대표 이미지** (img 또는 figure+figcaption)
5. **추천 이유** (한 줄)

---

## 카테고리 예시

자신의 취향에 맞게 카테고리를 구성하세요:

- 뉴스 / 정보 사이트
- 엔터테인먼트 / 동영상
- 쇼핑
- 개발 / 학습
- SNS / 커뮤니티
- 취미 관련

## 힌트

<details>
<summary>힌트 1: 이미지 링크 만들기</summary>

이미지를 클릭하면 해당 사이트로 이동하게 하려면 `<a>` 태그 안에 `<img>`를 넣으세요:

```html
<a href="https://www.naver.com" target="_blank" rel="noopener noreferrer">
  <img src="https://picsum.photos/seed/naver/300/200" alt="네이버 홈페이지 (클릭하면 이동)">
</a>
```
</details>

<details>
<summary>힌트 2: 앵커 링크 사용하기</summary>

페이지 상단에 목차를 만들고, 각 섹션으로 바로 이동할 수 있게 하세요:

```html
<!-- 상단 목차 -->
<h2>카테고리 목차</h2>
<ul>
  <li><a href="#news">뉴스 사이트</a></li>
  <li><a href="#entertainment">엔터테인먼트</a></li>
  <li><a href="#dev">개발 사이트</a></li>
</ul>

<!-- 각 섹션에 id 부여 -->
<h2 id="news">뉴스 사이트</h2>
...
<h2 id="entertainment">엔터테인먼트</h2>
...
<h2 id="dev">개발 사이트</h2>
```
</details>

<details>
<summary>힌트 3: 온라인 이미지 URL 사용하기</summary>

직접 이미지 파일이 없어도 온라인 이미지 서비스를 사용할 수 있습니다:

```html
<!-- Lorem Picsum: 랜덤 사진 제공 사이트 -->
<!-- https://picsum.photos/너비/높이 -->
<img src="https://picsum.photos/400/250" alt="랜덤 이미지">

<!-- seed를 지정하면 항상 같은 이미지 -->
<img src="https://picsum.photos/seed/naver/400/250" alt="고정된 랜덤 이미지">

<!-- 사이트 파비콘(아이콘) 이미지 사용하기 -->
<img src="https://www.google.com/favicon.ico" alt="구글 아이콘" width="32" height="32">
```
</details>

<details>
<summary>힌트 4: 전체 페이지 구조 예시</summary>

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>내 즐겨찾기 링크 모음</title>
</head>
<body>

  <h1>나의 즐겨찾기 모음</h1>
  <p>자주 방문하는 사이트들을 모아두었습니다.</p>

  <!-- 목차 -->
  <h2>목차</h2>
  <ul>
    <li><a href="#news">뉴스</a></li>
    <li><a href="#dev">개발</a></li>
    <li><a href="#hobby">취미</a></li>
  </ul>

  <hr>

  <!-- 뉴스 섹션 -->
  <h2 id="news">뉴스 사이트</h2>

  <h3>네이버 뉴스</h3>
  <figure>
    <a href="https://news.naver.com" target="_blank" rel="noopener noreferrer">
      <img src="이미지URL" alt="네이버 뉴스 (클릭 시 이동)" width="400">
    </a>
    <figcaption>네이버 뉴스 - 국내 최대 포털의 뉴스 서비스</figcaption>
  </figure>
  <p>
    국내 최대 포털 사이트 네이버의 뉴스 서비스입니다.
    다양한 카테고리의 뉴스를 한 곳에서 볼 수 있습니다.
  </p>
  <p>추천 이유: 가장 많은 뉴스를 빠르게 확인할 수 있어서</p>

  <p><a href="#">맨 위로</a></p>
  <hr>

  <!-- 나머지 섹션들... -->

</body>
</html>
```
</details>

| 항목 | 배점 |
|------|------|

| 이미지 삽입 (alt 속성 포함) | 20점 |
| figure + figcaption 사용 (3개 이상) | 15점 |
| 이미지 링크 (a 안에 img) | 10점 |
| 앵커 링크 및 목차 구성 | 15점 |
| mailto: 또는 tel: 링크 | 5점 |
| 카테고리 구성 및 내용 충실함 | 10점 |
| **합계** | **100점** |
