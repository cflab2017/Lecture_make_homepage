# 강의03 과제 — 언어 설정 Context

## 과제 목표

**한국어 / 영어 전환**이 가능한 언어 설정 Context를 구현하세요.  
Context를 통해 언어를 바꾸면 헤더와 버튼의 텍스트가 함께 변경되어야 합니다.

## 요구사항

### Context 구조

```jsx
// LanguageContext가 제공하는 값
{
  language: 'ko' | 'en',       // 현재 언어
  toggleLanguage: () => void,  // 언어 전환 함수
  t: (key) => string,          // 번역 함수
}
```

### 번역 텍스트

최소한 다음 키를 포함하세요:

| key | 한국어 (ko) | 영어 (en) |
|-----|------------|----------|
| `home` | 홈 | Home |
| `about` | 소개 | About |
| `contact` | 연락처 | Contact |
| `toggle` | 영어로 전환 | 한국어로 전환 |
| `welcome` | 안녕하세요! | Hello! |
| `description` | React 심화 강의입니다. | This is a React advanced course. |

### 컴포넌트 구조

```
App
 └─ LanguageProvider
     ├─ Header (t('home'), t('about'), t('contact'), 언어 전환 버튼)
     └─ MainContent (t('welcome'), t('description'))
```

### 동작 조건

1. `LanguageProvider`로 앱 전체를 감쌉니다.
2. 헤더의 네비게이션 메뉴 텍스트가 언어에 따라 바뀝니다.
3. 언어 전환 버튼을 클릭하면 언어가 전환됩니다.
4. `useLanguage()` 커스텀 훅을 만들어 Context를 추상화합니다.
5. Provider 없이 `useLanguage`를 사용하면 에러를 던집니다.

## 힌트

```jsx
// 번역 데이터를 객체로 정의
const translations = {
  ko: {
    home: '홈',
    // ...
  },
  en: {
    home: 'Home',
    // ...
  },
};

// t 함수 — 현재 언어에 해당하는 텍스트 반환
const t = (key) => translations[language][key] || key;
```
