# 과제 04 — 간단한 개인 홈페이지 서버

## 목표

Express를 사용해서 자신만의 개인 홈페이지를 서빙하는 서버를 만드세요.

---

## 요구사항

### 기본 기능

1. Express 서버를 만들고 포트 3000에서 실행합니다.
2. **3페이지 이상** HTML 파일을 제공합니다.
3. 모든 페이지에 **네비게이션 링크**가 있어야 합니다.
4. 각 페이지에 의미 있는 콘텐츠가 있어야 합니다.

### 필수 페이지

| 경로 | 내용 |
|------|------|
| `/` | 홈 — 자기소개, 인사말 |
| `/about` | 소개 — 기술 스택, 관심사 |
| `/portfolio` | 포트폴리오 — 프로젝트 목록 |
| `/contact` | 연락처 — 연락 방법 |

### 필수 기능

- `express.static()`으로 CSS 파일 제공
- 모든 페이지에 일관된 스타일 (공통 CSS 파일 사용)
- 존재하지 않는 경로 접근 시 404 페이지 반환

---

## 폴더 구조

```
과제/
├── app.js
└── public/
    ├── index.html
    ├── about.html
    ├── portfolio.html
    ├── contact.html
    ├── 404.html
    └── css/
        └── style.css
```

---

## 힌트

```js
// HTML 파일 응답
res.sendFile(path.join(__dirname, 'public', 'index.html'));

// 404 처리
app.use((req, res) => {
  res.status(404).sendFile(path.join(__dirname, 'public', '404.html'));
});
```

---

## 도전 과제

- 방문자 카운터 구현 (변수로 카운팅)
- `/api/pages` 엔드포인트로 페이지 목록 JSON 반환
- 현재 시간을 표시하는 `/api/time` 엔드포인트
