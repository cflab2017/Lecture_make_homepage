# 과제 01 — CLI 계산기

## 목표

`node calc.js 숫자1 연산자 숫자2` 형태로 실행하면 계산 결과를 출력하는 CLI 계산기를 만드세요.

---

## 요구사항

### 기본 기능

1. `process.argv`로 세 개의 인수(숫자1, 연산자, 숫자2)를 받습니다.
2. 지원하는 연산자: `+`, `-`, `*`, `/`
3. 결과를 깔끔하게 출력합니다.

### 실행 예시

```bash
node calc.js 10 + 20
# 출력: 10 + 20 = 30

node calc.js 100 - 37
# 출력: 100 - 37 = 63

node calc.js 6 * 7
# 출력: 6 * 7 = 42

node calc.js 15 / 4
# 출력: 15 / 4 = 3.75
```

### 예외 처리 (도전)

- 인수가 3개 미만이면 사용법 안내 메시지 출력
- 0으로 나누면 오류 메시지 출력
- 지원하지 않는 연산자면 오류 메시지 출력
- 숫자가 아닌 값이 입력되면 오류 메시지 출력
