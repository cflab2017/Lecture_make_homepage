# 과제 05 — 주소록 앱

## 과제 목표

객체 배열, 구조분해할당, 전개 연산자를 활용하여 **주소록 앱**을 만드세요.

---

## 요구사항

### 데이터 구조

각 연락처는 다음 속성을 가진 객체로 구성합니다:

```javascript
{
  id: 1,
  name: "홍길동",
  phone: "010-1234-5678",
  email: "hong@example.com"
}
```

### 기능

1. **추가**: 이름, 전화번호, 이메일을 입력하고 추가
2. **삭제**: 각 연락처의 삭제 버튼으로 삭제
3. **검색**: 이름 또는 이메일로 실시간 검색 (filter 사용)
4. **상세 보기**: 연락처 클릭 시 전체 정보 표시 (find + 구조분해할당 사용)

### 초기 데이터

```javascript
const contacts = [
  { id: 1, name: "홍길동", phone: "010-1111-2222", email: "hong@example.com" },
  { id: 2, name: "이순신", phone: "010-3333-4444", email: "lee@example.com" },
  { id: 3, name: "강감찬", phone: "010-5555-6666", email: "kang@example.com" },
];
```

### 검증 조건

- 이름은 **필수** 입력입니다.
- 전화번호는 `010-XXXX-XXXX` 형식을 권장합니다. (힌트: `000-0000-0000` 패턴)
- 이메일에 `@` 문자가 포함되어야 합니다.
- 이름이 중복된 연락처는 추가할 수 없습니다.

## 구현 방법 가이드

### 전개 연산자로 연락처 추가

```javascript
// 기존 배열을 복사하면서 새 항목 추가
contacts = [...contacts, { id: nextId++, name, phone, email }];
```

### 구조분해할당으로 폼 데이터 읽기

```javascript
const { name, phone, email } = getFormValues(); // 함수 반환값 구조분해
```

### filter로 검색

```javascript
const results = contacts.filter(c =>
  c.name.includes(keyword) || c.email.includes(keyword)
);
```
