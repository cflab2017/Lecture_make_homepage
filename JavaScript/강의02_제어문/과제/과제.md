# 과제 02 — 구구단 출력기

## 과제 목표

반복문과 조건문을 활용하여 **구구단 출력기**를 만드세요.

---

## 요구사항

### 기능

1. 1~9단 중 하나를 선택하는 **드롭다운** (`<select>`)을 만듭니다.
2. "출력하기" 버튼을 클릭하면 선택한 단의 구구단을 화면에 출력합니다.
3. **"전체 출력"** 버튼을 클릭하면 2단부터 9단까지 모두 출력합니다.

### 출력 형식 (예시)

**단일 단 (5단 선택 시)**
```
── 5단 ──
5 × 1 = 5
5 × 2 = 10
5 × 3 = 15
...
5 × 9 = 45
```

**전체 출력 (2~9단)**
```
── 2단 ──    ── 3단 ──    ...
2 × 1 = 2    3 × 1 = 3
...
```

---

## 조건

- `for` 반복문을 사용하여 구구단을 계산합니다.
- 1단은 드롭다운에 포함하되, 선택 시 "1단은 의미가 없어요! 2~9단을 선택해주세요." 메시지를 출력합니다.
- **전체 출력** 시 2~9단을 가로로 나열합니다. (CSS Grid 또는 flex 활용)
- 결과는 기존 내용을 **지우고** 새로 출력합니다.

---

## 힌트

```javascript
// 중첩 for 문 예시
for (let dan = 2; dan <= 9; dan++) {
  for (let i = 1; i <= 9; i++) {
    console.log(`${dan} × ${i} = ${dan * i}`);
  }
}

// 드롭다운 값 가져오기
const selected = parseInt(document.getElementById('danSelect').value);
```
