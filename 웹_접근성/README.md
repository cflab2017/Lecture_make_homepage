# 웹 접근성 과목

## 과목 소개

장애인, 고령자 등 **모든 사람이 웹을 사용할 수 있도록** 만드는 방법을 학습합니다.  
국제 표준인 **WCAG 2.1** 기준을 따르며, 실제 코드에 적용하는 방법을 다룹니다.

**선수 조건:** HTML + CSS 기초

---

## 강의 목록

| 강의 | 주제 | 핵심 내용 |
|------|------|-----------|
| 강의01 | 접근성 개념 | WCAG 4원칙, 장애 유형, 테스트 도구 |
| 강의02 | 시맨틱과 ARIA | ARIA 속성, 키보드 내비게이션, focus 관리 |
| 강의03 | 폼 접근성 | label-input 연결, 에러 메시지, 색상 대비 |
| 강의04 | 이미지/미디어 | alt 텍스트, 비디오 자막, prefers-reduced-motion |

---

## 학습 목표

- 웹 접근성의 필요성과 법적 의무를 이해한다
- WCAG 2.1의 4원칙을 설명할 수 있다
- 스크린리더와 키보드만으로 사이트를 탐색할 수 있게 만든다
- Lighthouse로 접근성 점수를 측정하고 개선할 수 있다
- ARIA 속성을 올바르게 사용할 수 있다

---

## 접근성 테스트 도구

```
Lighthouse    - Chrome DevTools 내장, 접근성 점수 측정
axe DevTools  - Chrome/Firefox 확장 프로그램, 상세 진단
NVDA          - Windows 무료 스크린리더
VoiceOver     - macOS/iOS 내장 스크린리더
Color Oracle  - 색맹 시뮬레이션 도구
```

## 관련 법규 (한국)

- **장애인차별금지법 제21조**: 정보 접근에서의 차별 금지
- **한국형 웹 콘텐츠 접근성 지침(KWCAG) 2.1**: WCAG 2.1 기반 국내 표준
