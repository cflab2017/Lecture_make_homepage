# Lecture_make_homepage

HTML/CSS/JavaScript/TypeScript를 처음 배우는 완전 초보자부터 실무 진입까지를 아우르는
**웹 개발 종합 강의 시리즈**입니다. 총 **64강**, 약 80~120시간 분량.

---

## 전체 커리큘럼 구조

```
[시작하기 5강] ─→ [HTML 6강] ─→ [CSS 7강] ─→ [JavaScript 8강]
                                                     ↓
[웹 보안 4강] ←─ [웹 접근성 4강] ←─ [Git 협업 4강] ←─┤
                                                     ↓
                                           [Node.js 6강]
                                                     ↓
                                          [데이터베이스 6강]
                                                     ↓
                                          [TypeScript 6강]
                                                     ↓
                                          [React 심화 8강]
```

---

## 모듈 (과목) 안내

### 🚀 [시작하기](./시작하기/) — 5강
**먼저 여기서 시작하세요.** 첫 홈페이지를 배포까지 완성하는 입문 시리즈.
GitHub + Vercel로 5분 만에 인터넷에 공개된 내 홈페이지를 만듭니다.

### 📄 [HTML](./HTML/) — 6강
웹 문서의 뼈대를 만드는 마크업 언어.
기초 → 텍스트 → 링크/이미지 → 테이블 → 폼 → 시맨틱 HTML

### 🎨 [CSS](./CSS/) — 7강
웹 페이지를 아름답고 반응형으로.
선택자 → 색상/폰트 → 박스 모델 → 배경/테두리 → Flexbox → Grid → 반응형

### ⚡ [JavaScript](./JavaScript/) — 8강
웹에 동작과 상호작용을.
변수 → 제어문 → 함수 → 배열 → 객체 → DOM → 이벤트 → 비동기

### 🟢 [Node.js](./Node.js/) — 6강
서버 사이드 JavaScript와 REST API.
기초 → 내장 모듈 → npm → Express → REST API → 파일 DB

### 🔷 [TypeScript](./TypeScript/) — 6강
타입 안전한 JavaScript로 진화.
기초 → 타입 선언 → 인터페이스 → 클래스 → 제네릭 → React + TS

### ⚛️ [React 심화](./React_심화/) — 8강
현업 수준 React 컴포넌트 설계와 상태 관리.
컴포넌트 → useEffect → useContext → useReducer → 커스텀훅 → 성능 → Router → Zustand

### 🔀 [Git 협업](./Git_협업/) — 4강
혼자 쓰는 Git을 넘어 팀 협업으로.
브랜치 전략 → 충돌 해결 → PR 리뷰 → Git 심화

### 💾 [데이터베이스](./데이터베이스/) — 6강
데이터를 영구 저장하는 방법.
DB 개념 → SQL 기초 → SQL 심화 → Node+SQLite → Prisma → MongoDB

### ♿ [웹 접근성](./웹_접근성/) — 4강
모든 사람이 사용할 수 있는 웹.
접근성 개념 → ARIA → 폼 접근성 → 이미지/미디어

### 🔒 [웹 보안](./웹_보안/) — 4강
실무에서 반드시 알아야 할 보안.
XSS → CSRF/인증 → SQL 인젝션 → HTTPS/환경변수

## 학습 경로 가이드


### 🔴 실무 준비 (3주~1개월)
```
Git 협업 → 웹 보안 → 웹 접근성
```
취업·실무에 바로 필요한 협업·품질 기술.

## 각 강의 폴더 구조

모든 강의는 동일한 표준 구조를 따릅니다:

```
강의01_주제명/
├── README.md      ← 강의 본문 (개념 설명 + 코드 예제)
├── 예제/           ← 강의 중 따라하는 완성 소스코드
├── 과제/           ← 과제 문제 (요구사항·힌트)
│   └── 과제.md
└── 과제정답/       ← 정답 코드 (한국어 주석 포함)
```

> 💡 **과제 풀이 시**: 정답을 먼저 보지 마세요. 30분 이상 직접 시도한 뒤 비교하면 학습 효과가 극대화됩니다.
