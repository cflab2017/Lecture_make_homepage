# 과제01 — 댓글 입력창 XSS 방어

## 목표

innerHTML을 사용하는 취약한 코드를 안전한 코드로 변환합니다.

---

## 취약한 코드 (이 코드를 수정하세요)

```html
<!DOCTYPE html>
<html>
<head><title>댓글</title></head>
<body>
    <input id="name" type="text" placeholder="이름">
    <textarea id="comment" placeholder="댓글 내용"></textarea>
    <button onclick="addComment()">등록</button>
    <div id="comment-list"></div>

    <script>
        function addComment() {
            const name    = document.getElementById('name').value;
            const comment = document.getElementById('comment').value;

            // ⚠ 취약점: 사용자 입력을 innerHTML에 직접 삽입
            document.getElementById('comment-list').innerHTML +=
                `<div><b>${name}</b>: ${comment}</div>`;
        }
    </script>
</body>
</html>
```

---

## 테스트할 공격 페이로드 (방어 후 이 내용이 실행되지 않아야 함)

이름: <script>alert('이름 XSS')</script>

## 구현해야 할 방어 방법

1. `textContent` 사용 (innerHTML 대신)
2. 또는 `escapeHtml()` 함수를 만들어 이스케이프 처리
3. 두 방법 모두 구현하고 차이를 주석으로 설명
