# 과제01 — 내 페이지 접근성 감사

## 목표

Lighthouse를 사용하여 기존에 만든 웹 페이지의 접근성을 점검하고,  
발견된 문제점과 해결 방법을 기록합니다.

---

## 과제 절차

### 1. 점검할 페이지 선택

지금까지 만든 프로젝트 중 하나를 선택하세요.  
(없다면 아무 웹사이트를 선택해도 됩니다.)

### 2. Lighthouse로 접근성 점수 측정

```
1. Chrome에서 페이지 열기
2. F12 → Lighthouse 탭
3. Accessibility만 체크
4. Analyze page load 클릭
5. 점수 및 항목 스크린샷 촬영
```

### 3. 문제점 3가지 기록

발견된 접근성 문제 중 중요한 3가지를 선택하여 아래 형식으로 기록하세요.

```markdown
## 문제 1: [문제 항목 이름]

**발견 위치:** (어떤 HTML 요소에서 발견되었는지)

**문제 설명:** (왜 접근성 문제인지)

**현재 코드:**
```html
(문제가 있는 코드)
```

**해결 방법:** (어떻게 고쳐야 하는지)
```

---

```
