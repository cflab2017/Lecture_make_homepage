# 과제02 — 접근성 모달 다이얼로그

## 목표

키보드와 스크린리더를 완벽하게 지원하는 **모달 다이얼로그**를 구현합니다.

---

## 요구사항

### 접근성 모달의 필수 조건

1. **`role="dialog"`** — 다이얼로그 역할 부여
2. **`aria-modal="true"`** — 모달 영역임을 스크린리더에 알림
3. **`aria-labelledby`** — 모달 제목과 연결
4. **포커스 이동** — 모달이 열리면 첫 번째 포커스 가능 요소로 이동
5. **Focus Trap** — 모달이 열린 동안 포커스가 모달 밖으로 나가지 않음
6. **Escape 닫기** — ESC 키로 모달 닫기
7. **포커스 복귀** — 모달이 닫히면 모달을 열었던 버튼으로 포커스 복귀
8. **배경 비활성화** — 모달 뒤의 콘텐츠는 `aria-hidden="true"` 처리

---

## 구현할 기능

```html
<!-- 이 버튼을 클릭하면 모달이 열림 -->
<button id="open-modal">주문 확인하기</button>

<!-- 모달 구조 -->
<div role="dialog" aria-modal="true" aria-labelledby="modal-title">
    <h2 id="modal-title">주문을 완료하시겠습니까?</h2>
    <p>주문 내역: 노트북 1개 / 총 금액: 1,500,000원</p>
    <button id="confirm-btn">주문 확인</button>
    <button id="cancel-btn">취소</button>
</div>
```

## Focus Trap 구현 힌트

```js
// 모달 내부의 포커스 가능한 요소들
const focusableSelectors = [
    'a[href]', 'button:not([disabled])',
    'input:not([disabled])', 'select',
    'textarea', '[tabindex]:not([tabindex="-1"])'
].join(', ');

function trapFocus(modal) {
    const focusableElements = modal.querySelectorAll(focusableSelectors);
    const firstFocusable = focusableElements[0];
    const lastFocusable = focusableElements[focusableElements.length - 1];

    modal.addEventListener('keydown', (e) => {
        if (e.key !== 'Tab') return;

        if (e.shiftKey) {
            // Shift+Tab: 첫 번째 요소에서 마지막으로 이동
            if (document.activeElement === firstFocusable) {
                e.preventDefault();
                lastFocusable.focus();
            }
        } else {
            // Tab: 마지막 요소에서 첫 번째로 이동
            if (document.activeElement === lastFocusable) {
                e.preventDefault();
                firstFocusable.focus();
            }
        }
    });
}
```
