# 과제 — 나만의 배지 만들기

## 목표

CSS의 클래스 선택자와 ID 선택자를 활용하여 **서로 다른 스타일의 배지(Badge) 3종**을 만듭니다.

---

## 요구사항

### 1. HTML 파일 (`index.html`)

- `index.html`과 `style.css`를 **같은 폴더**에 만들고, 외부 CSS 파일로 연결하세요.
- 아래 구조를 참고하여 배지 3개를 만드세요.

```html
<div class="badge-container">
  <span class="badge badge-success">성공</span>
  <span class="badge badge-warning">경고</span>
  <span id="badge-special" class="badge">특별</span>
</div>
```

### 2. CSS 파일 (`style.css`)

다음 조건을 모두 만족하는 스타일을 작성하세요.

| 선택자 | 적용 내용 |
|--------|-----------|
| `body` | 배경색, 글꼴, 중앙 정렬 설정 |
| `.badge` | 모든 배지 공통 스타일 (padding, border-radius, font-weight 등) |
| `.badge-success` | 초록 계열 배지 |
| `.badge-warning` | 노란/주황 계열 배지 |
| `#badge-special` | 보라 또는 파랑 계열의 특별한 배지 (ID 선택자 사용) |
| `.badge:hover` | 마우스를 올리면 약간 커지거나 색이 변하는 효과 |

### 3. 추가 도전 (선택사항)

- 배지 아래에 각 배지가 어떤 선택자로 만들어졌는지 설명하는 텍스트를 추가하세요.
- 배지를 클릭했을 때 (`a:active`) 다른 색상이 적용되도록 만들어 보세요.
- 형제 선택자(`+`)를 사용하여 두 번째 배지에만 왼쪽 여백을 추가해 보세요.

---

## 제출 파일

```
내폴더/
├── index.html
└── style.css
```

---

## 체크리스트

- [ ] 인라인 CSS를 사용하지 않았다
- [ ] `<link>` 태그로 외부 CSS 파일을 연결했다
- [ ] `.badge` 공통 클래스가 존재한다
- [ ] 배지 3종이 서로 다른 색상을 가진다
- [ ] `#badge-special` ID 선택자를 사용했다
- [ ] `:hover` 가상 클래스를 적용했다
- [ ] 브라우저에서 오류 없이 열린다
