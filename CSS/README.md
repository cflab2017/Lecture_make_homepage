# CSS 강의 자료

웹 페이지를 아름답게 만드는 CSS(Cascading Style Sheets)의 핵심 개념을 단계별로 학습합니다.

---

## 선수 조건

- HTML 기초 (태그, 속성, 문서 구조)
- 텍스트 편집기 사용법 (VS Code 권장)
- 웹 브라우저 개발자 도구 기초

> HTML 기초 강의를 먼저 수강하지 않은 경우, `../HTML/` 폴더의 강의를 먼저 완료해 주세요.

---

## 강의 목록

| 강의 | 주제 | 핵심 개념 |
|------|------|-----------|
| 강의 01 | CSS 기초 | 연결 방법, 선택자, 우선순위, 상속 |
| 강의 02 | 색상 & 폰트 | 색상 표현법, Google Fonts, CSS 변수 |
| 강의 03 | 박스 모델 | content/padding/border/margin, box-sizing |
| 강의 04 | 배경 & 테두리 | background, border-radius, box-shadow, transition |
| 강의 05 | Flexbox | flex 배치, justify-content, align-items |
| 강의 06 | Grid | grid-template, fr 단위, grid-template-areas |
| 강의 07 | 반응형 웹 | @media 쿼리, mobile-first, breakpoint |

---

## 학습 방법

1. 각 강의 폴더의 `README.md`를 읽으며 개념을 이해합니다.
2. `예제/` 폴더의 파일을 브라우저에서 열어 직접 확인합니다.
3. 코드를 수정하며 속성 변화를 관찰합니다.
4. `과제/과제.md`를 읽고 직접 구현합니다.
5. 막히면 `과제정답/`을 참고합니다.

---

## 폴더 구조

```
CSS/
├── README.md               ← 현재 파일
├── 강의01_CSS_기초/
│   ├── README.md
│   ├── 예제/               ← index.html + style.css
│   ├── 과제/               ← 과제.md
│   └── 과제정답/           ← index.html + style.css (주석 포함)
├── 강의02_색상_폰트/
├── 강의03_박스_모델/
├── 강의04_배경_테두리/
├── 강의05_Flexbox/
├── 강의06_Grid/
└── 강의07_반응형/
```
