Flexbox vs Grid

두 레이아웃 시스템의 장단점을 비교하고 올바른 선택 기준을 알아봅니다. Grid는 2차원, Flexbox는 1차원 레이아웃에 최적화되어 있습니다.

실무에서는 두 기술을 함께 사용하는 것이 가장 효과적입니다.

TIPS

fr 단위란?

남은 공간을 비율로 나누는 Grid 전용 단위입니다. 반응형 레이아웃에 필수입니다.
