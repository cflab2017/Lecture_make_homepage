2. 폰트 패밀리 비교

Noto Sans KR: 한글 웹에서 가장 많이 사용되는 폰트입니다. 가독성이 뛰어납니다.

Playfair Display: Elegant serif font for headings and display text.

Monospace: 코드 블록에 자주 사용됩니다. const greeting = "Hello";

Serif: 전통적인 명조 계열 폰트입니다. 인쇄물 느낌을 줍니다.

3. 폰트 굵기 (font-weight)

300 — Light: 얇고 가벼운 텍스트

400 — Normal: 일반적인 본문 텍스트

700 — Bold: 굵고 강조된 텍스트

900 — Black: 매우 굵은 텍스트

4. 글자 크기 (font-size)

12px — 매우 작은 텍스트 (주석, 캡션)

14px — 작은 텍스트

16px — 기본 크기 (본문 권장)

20px — 약간 큰 텍스트 (소제목)

28px — 큰 텍스트 (제목)

5. 줄 간격 & 글자 간격

line-height: 1.0 (좁음)

한국어 텍스트는 줄 간격이 중요합니다. 좁은 줄 간격은 읽기 불편합니다. 이처럼 텍스트가 빽빽하게 느껴집니다.

line-height: 1.8 (적당)

한국어 텍스트는 줄 간격이 중요합니다. 적당한 줄 간격은 가독성을 크게 높입니다. 이처럼 텍스트가 편안하게 읽힙니다.

letter-spacing: -1px — 글자가 촘촘하게

letter-spacing: 0 — 기본값

letter-spacing: 3px — 글자가 넓게 펼쳐짐

6. 텍스트 꾸미기

text-decoration: underline — 밑줄

text-decoration: line-through — 취소선

text-transform: uppercase — 모두 대문자

text-transform: capitalize — 첫 글자 대문자

7. CSS 변수 활용

아래 색상들은 모두 :root에 선언된 CSS 변수로 적용되었습니다.

--color-primary
--color-secondary
--color-accent

변수 하나를 바꾸면 이 색상이 적용된 모든 요소가 한번에 바뀝니다!
