# 과제 — 이미지 카드 갤러리

## 목표

배경 이미지, 그림자, hover 전환 효과를 활용하여  
마우스를 올리면 반응하는 **이미지 카드 갤러리**를 만듭니다.

---

## 요구사항

### 1. 카드 3개 만들기

각 카드는 다음 구조로 만드세요.

```html
<div class="gallery">
  <div class="gallery-card">
    <div class="card-image"></div>
    <div class="card-info">
      <h3>제목</h3>
      <p>설명 텍스트</p>
    </div>
  </div>
  <!-- 카드 2, 3 반복 -->
</div>
```

### 2. 필수 CSS 속성

| 속성 | 요구사항 |
|------|----------|
| `background-image` | 각 카드마다 다른 그라데이션 배경 (`linear-gradient`) |
| `background-size` | `cover` 적용 |
| `border-radius` | 카드 전체에 둥근 모서리 |
| `box-shadow` | 기본 상태 그림자 설정 |
| `transition` | 호버 시 부드럽게 변하도록 설정 |
| `:hover` | 마우스 올리면 그림자 강해지고 살짝 올라오는 효과 |

### 3. 카드 이미지 높이

- `.card-image` 높이: 200px 이상
- 배경 이미지(그라데이션)로 채우기

### 4. 추가 도전 (선택사항)

- 카드에 `overflow: hidden`을 적용하여 이미지에 `border-radius`가 적용되도록 하세요.
- 호버 시 카드 이미지가 약간 확대(`scale`)되는 효과를 추가하세요.
- `opacity`를 활용한 오버레이 효과를 추가하세요.

---

## 체크리스트

- [ ] 카드 3개가 각각 다른 그라데이션 배경을 가진다
- [ ] `box-shadow`가 적용되었다
- [ ] 호버 시 그림자가 변하는 `transition` 효과가 있다
- [ ] 카드 모서리가 `border-radius`로 둥글다
- [ ] 인라인 CSS를 사용하지 않았다

## 힌트

```css
.gallery {
  display: flex;
  gap: 20px;
}

.gallery-card {
  flex: 1;
  border-radius: 12px;
  overflow: hidden;
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.1);
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.gallery-card:hover {
  transform: translateY(-8px);
  box-shadow: 0 12px 30px rgba(0, 0, 0, 0.2);
}

.card-image {
  height: 200px;
  background: linear-gradient(135deg, #667eea, #764ba2);
  background-size: cover;
  background-position: center;
}
```
