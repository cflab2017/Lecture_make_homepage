박스 모델 시각화 데모

각 영역(content / padding / border / margin)을 색상으로 구분하여 확인합니다.

1. 박스 모델 구조

margin
border
padding
content

2. box-sizing 비교

두 박스 모두 width: 300px이지만 실제 크기가 다릅니다.

content-box (기본값)

width: 300px
padding: 20px
border: 4px

실제 너비: 348px

border-box (권장)

width: 300px
padding: 20px
border: 4px

실제 너비: 300px

3. display 속성 비교

display: block

블록 요소는 한 줄을 독차지합니다.

블록 A
블록 B

display: inline

인라인 요소는 텍스트처럼 나란히 배치됩니다. (width, height 무시)

인라인 A 인라인 B 인라인 C

display: inline-block

인라인처럼 나란히 배치되면서 width, height 설정이 가능합니다.

인라인블록 A 인라인블록 B 인라인블록 C

4. overflow 속성

overflow: visible

내용이 넘치면 박스 밖으로 튀어나옵니다. 기본값입니다. 이 텍스트는 박스를 넘어갑니다!

overflow: hidden

내용이 넘치면 잘려서 보이지 않습니다. 이 텍스트는 박스를 넘어가도 숨겨집니다!

overflow: auto

내용이 넘치면 스크롤이 생깁니다. 이 텍스트는 길어서 스크롤이 필요합니다. 아래로 스크롤해 보세요. 더 많은 내용이 있습니다!

5. margin 겹침 현상

아래 두 박스의 margin-bottom/top은 각각 30px이지만,
실제 간격은 30px입니다 (합산 60px이 아님).

margin-bottom: 30px
margin-top: 30px

6. margin auto로 수평 중앙 배치

margin: 0 auto
width: 200px
