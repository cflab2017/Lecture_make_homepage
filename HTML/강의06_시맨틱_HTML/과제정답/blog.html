소개

안녕하세요! 저는 웹 개발에 깊은 관심을 갖고 공부하고 있는 홍길동입니다. HTML부터 시작하여 CSS, JavaScript를 차례로 배우고 있으며, 언젠가 사람들에게 실제로 도움이 되는 웹 서비스를 만드는 것이 꿈입니다.

현재는 웹의 기초인 HTML을 열심히 공부하고 있습니다. 코딩을 배우면서 작은 것들을 만들어가는 과정이 너무 즐겁습니다. 앞으로 CSS와 JavaScript도 익혀서 더 완성도 있는 웹 페이지를 만들어 보겠습니다!

자기소개 웹 페이지

제작 기간: 2026년 3월 ~ 2026년 4월

HTML을 처음 배우고 만든 자기소개 페이지입니다. DOCTYPE 선언부터 head, body 구조, h1~h6, p, ul, ol 태그를 활용했습니다. CSS 없이 순수 HTML만으로 내용을 구성하는 연습을 했습니다.

사용 기술: HTML5

즐겨찾기 링크 모음 페이지

제작 기간: 2026년 4월 ~ 2026년 4월

a 태그, img 태그, figure/figcaption을 활용하여 만든 링크 모음 페이지입니다. 외부 링크 보안 처리(rel="noopener noreferrer"), 이미지 링크, 앵커 링크를 이용한 페이지 내 이동 기능을 구현했습니다.

사용 기술: HTML5 (a, img, figure, figcaption)

주간 수업 시간표

제작 기간: 2026년 5월 ~ 2026년 5월

HTML 테이블로 만든 주간 수업 시간표입니다. thead, tbody, tfoot 으로 테이블을 구조화하고, colspan을 사용하여 점심시간을 가로로 병합하고, rowspan으로 과목을 세로로 병합하는 기법을 연습했습니다.

사용 기술: HTML5 (table, thead, tbody, tfoot, colspan, rowspan)

회원가입 & 설문조사 폼

제작 기간: 2026년 5월

text, email, password, number, date, radio, checkbox, file, range 등 다양한 input 타입을 모두 활용하여 만든 회원가입 폼입니다. fieldset과 legend로 관련 입력 요소를 그룹화하고, required, placeholder 등 검증 속성도 적용했습니다.

사용 기술: HTML5 (form, input, select, textarea, fieldset, label)
