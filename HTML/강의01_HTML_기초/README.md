# 강의 01 - HTML 기초

> HTML 문서의 뼈대를 이루는 기본 구조와 자주 쓰이는 태그들을 배웁니다.

---

## 1. HTML이란?

**HTML**(HyperText Markup Language)은 웹 페이지의 **구조와 내용**을 기술하는 언어입니다.  
프로그래밍 언어가 아니라 **마크업 언어**로, 태그(tag)를 사용해 문서의 구조를 표현합니다.

```
<태그이름>내용</태그이름>
```

- 여는 태그(`<태그>`)와 닫는 태그(`</태그>`)가 쌍을 이룹니다.
- 일부 태그는 닫는 태그 없이 단독으로 사용됩니다 (예: `<br>`, `<img>`, `<input>`)

---

## 2. DOCTYPE 선언

```html
<!DOCTYPE html>
```

- HTML 문서의 **첫 번째 줄**에 반드시 작성합니다.
- 브라우저에게 "이 문서는 HTML5 형식입니다"라고 알려주는 선언입니다.
- `<!DOCTYPE html>`은 HTML5 표준 선언으로, 대소문자를 구분하지 않습니다.

---

## 3. HTML 문서의 기본 구조

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
  </head>
  <body>
    <!-- 여기에 실제 화면에 보이는 내용을 작성합니다 -->
    <h1>안녕하세요!</h1>
    <p>첫 번째 HTML 문서입니다.</p>
  </body>
</html>
```

| 태그 | 역할 |
|------|------|
| `<html lang="ko">` | 문서 전체를 감싸는 최상위 태그. `lang` 속성으로 언어를 지정 |
| `<head>` | 브라우저에 보이지 않는 **문서 정보**를 담는 영역 |
| `<body>` | 브라우저 화면에 **실제로 표시**되는 내용 |

---

## 4. 메타태그 (Meta Tags)

메타태그는 `<head>` 안에 작성하며, 문서에 대한 **부가 정보**를 제공합니다.

```html
<!-- 문자 인코딩 설정: 한글이 깨지지 않도록 UTF-8로 설정 -->
<meta charset="UTF-8">

<!-- 반응형 웹을 위한 뷰포트 설정 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 검색 엔진을 위한 페이지 설명 -->
<meta name="description" content="이 페이지는 HTML 기초를 배우는 예제 페이지입니다.">

<!-- 검색 엔진을 위한 키워드 -->
<meta name="keywords" content="HTML, 웹, 기초, 강의">

<!-- 작성자 정보 -->
<meta name="author" content="홍길동">

<!-- 브라우저 탭에 표시되는 제목 -->
<title>나의 첫 HTML 페이지</title>
```

## 5. 제목 태그 (h1 ~ h6)

HTML에는 6단계의 제목 태그가 있습니다. 숫자가 클수록 글자가 작아집니다.

```html
<h1>1단계 제목 - 가장 크고 중요한 제목 (페이지당 1개 권장)</h1>
<h2>2단계 제목 - 주요 섹션의 제목</h2>
<h3>3단계 제목 - 소제목</h3>
<h4>4단계 제목</h4>
<h5>5단계 제목</h5>
<h6>6단계 제목 - 가장 작은 제목</h6>
```

> **주의**: 제목 태그는 **글자 크기 조절**이 아닌 **문서의 계층 구조** 표현에 사용합니다.  
> 글자 크기는 CSS로 조절하세요.

---

## 6. 단락 태그 (p)

```html
<p>이것은 첫 번째 단락입니다. 단락은 앞뒤로 자동으로 여백이 생깁니다.</p>
<p>이것은 두 번째 단락입니다. p 태그 안에서는 줄바꿈을 해도 브라우저에서는 하나의 공백으로 처리됩니다.</p>
```

## 7. div와 span

| 태그 | 특성 | 용도 |
|------|------|------|
| `<div>` | **블록 레벨** 요소 (줄 전체를 차지) | 큰 영역을 묶을 때 |
| `<span>` | **인라인** 요소 (내용만큼만 차지) | 텍스트 일부를 묶을 때 |

```html
<!-- div: 블록 레벨 - 줄바꿈이 자동으로 생김 -->
<div>이 영역은 한 줄 전체를 차지합니다.</div>
<div>다음 div는 새 줄에서 시작합니다.</div>

<!-- span: 인라인 - 텍스트 흐름 안에서 사용 -->
<p>이 문장에서 <span>이 부분만</span> 별도로 스타일을 줄 수 있습니다.</p>
```

## 8. 목록 태그

### 순서 없는 목록 (ul - Unordered List)

```html
<ul>
  <li>사과</li>
  <li>바나나</li>
  <li>딸기</li>
</ul>
```

결과:
- 사과
- 바나나
- 딸기

### 순서 있는 목록 (ol - Ordered List)

```html
<ol>
  <li>재료를 준비한다</li>
  <li>물을 끓인다</li>
  <li>재료를 넣는다</li>
  <li>5분 후 완성!</li>
</ol>
```

결과:
1. 재료를 준비한다
2. 물을 끓인다
3. 재료를 넣는다
4. 5분 후 완성!

### 중첩 목록

```html
<ul>
  <li>과일
    <ul>
      <li>사과</li>
      <li>바나나</li>
    </ul>
  </li>
  <li>채소
    <ul>
      <li>당근</li>
      <li>시금치</li>
    </ul>
  </li>
</ul>
```

## 정리

| 태그 | 설명 |
|------|------|
| `<!DOCTYPE html>` | HTML5 문서 선언 |
| `<html>` | 문서 최상위 요소 |
| `<head>` | 문서 메타 정보 |
| `<body>` | 화면에 표시되는 내용 |
| `<meta>` | 문서 부가 정보 |
| `<title>` | 브라우저 탭 제목 |
| `<h1>~<h6>` | 제목 (계층 구조) |
| `<p>` | 단락 |
| `<div>` | 블록 레벨 컨테이너 |
| `<span>` | 인라인 컨테이너 |
| `<ul>` | 순서 없는 목록 |
| `<ol>` | 순서 있는 목록 |
| `<li>` | 목록 항목 |
