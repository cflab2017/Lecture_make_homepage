# 과제 01 — 온도 단위 변환기

## 과제 목표

변수, 타입, 형변환, 템플릿 리터럴을 활용하여 **온도 단위 변환기**를 만드세요.

---

## 요구사항

### 기능

1. 사용자가 온도 값을 입력하는 **텍스트 입력창** (`<input type="number">`)을 만듭니다.
2. 변환 기준 단위를 선택하는 **드롭다운** (`<select>`)을 만듭니다.
   - 선택지: 섭씨(°C), 화씨(°F), 켈빈(K)
3. "변환하기" **버튼**을 클릭하면 나머지 두 단위로 변환한 결과를 화면에 출력합니다.

### 변환 공식

| 변환 | 공식 |
|------|------|
| 섭씨 → 화씨 | `°F = (°C × 9/5) + 32` |
| 섭씨 → 켈빈 | `K = °C + 273.15` |
| 화씨 → 섭씨 | `°C = (°F - 32) × 5/9` |
| 화씨 → 켈빈 | `K = (°F - 32) × 5/9 + 273.15` |
| 켈빈 → 섭씨 | `°C = K - 273.15` |
| 켈빈 → 화씨 | `°F = (K - 273.15) × 9/5 + 32` |

### 출력 형식 (예시)

```
입력값: 100 °C

🌡️ 섭씨(°C): 100.00 °C
🌡️ 화씨(°F): 212.00 °F
🌡️ 켈빈(K): 373.15 K
```

---

## 조건

- 모든 코드를 **하나의 HTML 파일**에 작성합니다. (별도 JS 파일 불필요)
- 결과는 `innerHTML` 또는 `textContent`를 사용해 화면에 동적으로 출력합니다.
- 숫자 결과는 소수점 **2자리**까지 표시합니다. (`toFixed(2)` 활용)
- 입력값이 비어 있거나 숫자가 아니면 "올바른 숫자를 입력해주세요." 메시지를 출력합니다.
- 켈빈 값이 **0 미만**이면 "켈빈은 0 이상이어야 합니다." 경고를 출력합니다.
- 템플릿 리터럴을 사용하여 결과 문자열을 만드세요.
