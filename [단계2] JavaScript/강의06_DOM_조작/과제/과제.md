# 과제 06 — 색상 팔레트 생성기

## 과제 목표

`createElement`, `classList`, `dataset`을 활용하여 **색상 팔레트 생성기**를 만드세요.

---

## 요구사항

### 기능

1. **색상 박스 추가**: 버튼 클릭 시 랜덤 배경색을 가진 박스를 화면에 추가합니다.
2. **HEX 코드 표시**: 박스를 클릭하면 해당 색상의 HEX 코드(`#RRGGBB`)를 하단에 표시합니다.
3. **클립보드 복사**: HEX 코드를 클릭하면 클립보드에 복사됩니다.
4. **선택 상태**: 현재 선택된 박스에 테두리를 표시합니다. (`classList.toggle` 또는 `classList.add/remove` 활용)
5. **박스 삭제**: 각 박스에 삭제 버튼을 추가합니다.

### 출력 형식

```
[색상 박스 추가] 버튼

┌─────────────────────────────────────┐
│  #FF5733  #3498DB  #27AE60  ... (박스들) │
└─────────────────────────────────────┘

선택된 색상: #FF5733  (클릭하여 복사)
```

---

## 조건

- **랜덤 HEX 색상** 생성:
  ```javascript
  function randomHex() {
    return '#' + Math.floor(Math.random() * 0xFFFFFF).toString(16).padStart(6, '0');
  }
  ```
- `dataset.color`에 HEX 값을 저장하고, 클릭 시 읽어 표시합니다.
- `classList.add('selected')`로 선택 상태를 표시합니다.
- 이전에 선택된 박스는 `classList.remove('selected')`로 선택 해제합니다.
- `createElement`로 박스 요소를 생성합니다.

---

## 힌트

```javascript
// 랜덤 HEX 생성
const color = '#' + Math.floor(Math.random() * 0xFFFFFF).toString(16).padStart(6, '0');

// 박스 생성
const box = document.createElement('div');
box.className = 'color-box';
box.style.backgroundColor = color;
box.dataset.color = color; // dataset에 색상 저장

// 클립보드 복사
navigator.clipboard.writeText(color).then(() => {
  alert(`${color} 복사됨!`);
});

// 선택 상태 관리
document.querySelectorAll('.color-box').forEach(b => b.classList.remove('selected'));
box.classList.add('selected');
```
