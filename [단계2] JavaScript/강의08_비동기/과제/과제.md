# 과제 08 — 날씨 앱

## 과제 목표

`fetch` API, `async/await`, `try/catch`를 활용하여 **실시간 날씨 앱**을 만드세요.

---

## 요구사항

### 사용할 API

**Open-Meteo** (무료, API 키 불필요)

```
GET https://api.open-meteo.com/v1/forecast?latitude=37.5665&longitude=126.9780&current=temperature_2m,weather_code
```

응답 예시:
```json
{
  "current": {
    "time": "2024-01-15T12:00",
    "temperature_2m": 3.2,
    "weather_code": 0
  }
}
```

### Weather Code 설명

| 코드 | 날씨 | 이모지 |
|------|------|--------|
| 0 | 맑음 | ☀️ |
| 1, 2, 3 | 대체로 맑음 / 흐림 | ⛅ |
| 45, 48 | 안개 | 🌫️ |
| 51, 53, 55 | 이슬비 | 🌦️ |
| 61, 63, 65 | 비 | 🌧️ |
| 71, 73, 75 | 눈 | 🌨️ |
| 80, 81, 82 | 소나기 | 🌩️ |
| 95, 96, 99 | 뇌우 | ⛈️ |

### 기능

1. 페이지 로드 시 자동으로 날씨 데이터를 가져옵니다.
2. "새로고침" 버튼 클릭 시 최신 데이터를 다시 가져옵니다.
3. **로딩 상태**: 데이터를 가져오는 동안 로딩 메시지를 표시합니다.
4. **에러 처리**: 네트워크 오류 시 에러 메시지와 재시도 버튼을 표시합니다.
5. 마지막 업데이트 시간을 표시합니다.

### 출력 형식

```
서울 현재 날씨 🌤

☀️ 맑음
🌡️ 3.2°C

마지막 업데이트: 오후 3:30:15
```

---

## 조건

- `async/await` 문법을 사용합니다.
- `try/catch/finally` 블록으로 에러를 처리합니다.
- 로딩 중에는 버튼을 비활성화합니다.
- `weather_code`를 `getWeatherInfo(code)` 함수로 변환합니다.
- 온도는 소수점 1자리까지 표시합니다.

---

## 힌트

```javascript
// weather_code에 따른 날씨 정보 함수
function getWeatherInfo(code) {
  if (code === 0) return { emoji: '☀️', desc: '맑음' };
  if (code <= 3)  return { emoji: '⛅', desc: '대체로 흐림' };
  if (code <= 48) return { emoji: '🌫️', desc: '안개' };
  if (code <= 55) return { emoji: '🌦️', desc: '이슬비' };
  if (code <= 65) return { emoji: '🌧️', desc: '비' };
  if (code <= 75) return { emoji: '🌨️', desc: '눈' };
  if (code <= 82) return { emoji: '🌩️', desc: '소나기' };
  return { emoji: '⛈️', desc: '뇌우' };
}

// fetch + async/await 기본 패턴
async function getWeather() {
  const url = 'https://api.open-meteo.com/v1/forecast?latitude=37.5665&longitude=126.9780&current=temperature_2m,weather_code';
  const response = await fetch(url);
  const data = await response.json();
  return data.current; // { time, temperature_2m, weather_code }
}
```
