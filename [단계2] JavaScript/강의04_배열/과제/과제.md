# 과제 04 — 영화 즐겨찾기

## 과제 목표

배열 메서드(`push`, `filter`, `map`, `includes`)를 활용하여 **영화 즐겨찾기 앱**을 만드세요.

---

## 요구사항

### 기능

1. **영화 추가**: 영화 제목을 입력하고 "추가" 버튼 클릭 시 목록에 추가
2. **검색**: 검색창에 입력하면 실시간으로 제목을 포함한 영화만 필터링하여 표시
3. **삭제**: 각 영화 항목의 삭제 버튼으로 개별 삭제
4. **전체 삭제**: 버튼 클릭 시 목록 전체 삭제

### 초기 데이터

앱 시작 시 다음 영화가 목록에 있어야 합니다:

```javascript
const movies = [
  "인터스텔라",
  "기생충",
  "어벤져스: 엔드게임",
  "타이타닉",
  "라라랜드"
];
```

### 추가 조건

- 동일한 영화 제목이 이미 존재하면 "이미 추가된 영화입니다." 메시지를 출력합니다. (`includes` 사용)
- 빈 제목은 추가되지 않습니다.
- 검색창이 비어있으면 전체 목록을 표시합니다.
- 검색은 대소문자 구분 없이 동작합니다. (`.toLowerCase()` 활용)
- `map`을 사용하여 목록을 렌더링합니다.
- `filter`를 사용하여 검색과 삭제를 구현합니다.

---

## 힌트

```javascript
// 중복 확인 (includes)
if (movies.includes(newTitle)) { /* 중복 처리 */ }

// 검색 필터 (filter + includes + toLowerCase)
const filtered = movies.filter(m => 
  m.toLowerCase().includes(searchTerm.toLowerCase())
);

// 목록 렌더링 (map + join)
const html = filtered.map((movie, index) => `
  <li>${movie} <button onclick="deleteMovie(${index})">삭제</button></li>
`).join('');
```
