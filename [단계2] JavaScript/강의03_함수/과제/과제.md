# 과제 03 — 단위 변환기 함수 모음

## 과제 목표

함수 선언식, 표현식, 화살표 함수를 활용하여 **단위 변환기 함수 모음**을 만드세요.

---

## 요구사항

### 구현할 변환 함수

| 함수 이름 | 기능 | 공식 |
|-----------|------|------|
| `kgToLb(kg)` | 킬로그램 → 파운드 | `lb = kg × 2.20462` |
| `lbToKg(lb)` | 파운드 → 킬로그램 | `kg = lb / 2.20462` |
| `kmToMile(km)` | 킬로미터 → 마일 | `mile = km × 0.621371` |
| `mileToKm(mile)` | 마일 → 킬로미터 | `km = mile / 0.621371` |
| `celsiusToFahrenheit(c)` | 섭씨 → 화씨 | `°F = (°C × 9/5) + 32` |
| `fahrenheitToCelsius(f)` | 화씨 → 섭씨 | `°C = (°F - 32) × 5/9` |

> 추가: `cmToInch(cm)` (1인치 = 2.54cm), `inchToCm(inch)` 도 구현하면 가산점!

### UI 요구사항

1. 각 변환 그룹(무게, 거리, 온도)을 **섹션**으로 구분합니다.
2. 각 섹션에 **입력창**과 **변환 방향 버튼** 2개를 만듭니다.
   - 예: [kg → lb 변환] [lb → kg 변환]
3. 결과는 각 섹션 내에 출력합니다.
4. 결과는 소수점 **3자리**까지 표시합니다.

### 예시

```
[무게 변환]
입력: [    70    ] kg
[kg → lb 변환]  →  154.324 lb
[lb → kg 변환]  →  31.751 kg
```

---

## 조건

- 변환 함수는 **화살표 함수**로 작성합니다.
- 각 함수는 **한 줄** 화살표 함수로 작성합니다. (`=>` 뒤에 바로 반환값)
- 입력값이 비어 있거나 숫자가 아니면 에러 메시지를 출력합니다.
- 모든 코드를 하나의 HTML 파일에 작성합니다.

---

## 힌트

```javascript
// 한 줄 화살표 함수 예시
const kgToLb = kg => kg * 2.20462;
const lbToKg = lb => lb / 2.20462;

// 결과 소수점 3자리
const result = kgToLb(70).toFixed(3); // "154.323"

// 공통 변환 실행 함수 (콜백 활용)
function convert(inputId, resultId, convertFn, unit) {
  const value = parseFloat(document.getElementById(inputId).value);
  if (isNaN(value)) { /* 에러 처리 */ return; }
  const result = convertFn(value);
  document.getElementById(resultId).textContent = `${result.toFixed(3)} ${unit}`;
}
```
