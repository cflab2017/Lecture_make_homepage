# 과제 07 — 계산기 UI

## 과제 목표

이벤트 처리(`addEventListener`, `event.target`, `keydown`)를 활용하여 **계산기 UI**를 만드세요.

---

## 요구사항

### UI 구성

```
┌─────────────────────────────────┐
│             0               ← 화면  │
├────┬────┬────┬────┤
│ C  │ ±  │ %  │  ÷ │
├────┼────┼────┼────┤
│ 7  │ 8  │ 9  │  × │
├────┼────┼────┼────┤
│ 4  │ 5  │ 6  │  − │
├────┼────┼────┼────┤
│ 1  │ 2  │ 3  │  + │
├─────────┼────┼────┤
│    0    │  . │  = │
└─────────┴────┴────┘
```

### 버튼 기능

| 버튼 | 기능 |
|------|------|
| `0~9` | 숫자 입력 |
| `+`, `−`, `×`, `÷` | 사칙연산 |
| `=` | 계산 실행 |
| `C` | 초기화 (Clear) |
| `±` | 부호 전환 (양수 ↔ 음수) |
| `%` | 퍼센트 (÷ 100) |
| `.` | 소수점 (한 번만 입력 가능) |

### 키보드 지원

| 키보드 키 | 기능 |
|-----------|------|
| `0~9` | 숫자 입력 |
| `+`, `-`, `*`, `/` | 연산자 |
| `Enter` 또는 `=` | 계산 |
| `Backspace` | 마지막 숫자 삭제 |
| `Escape` | 초기화 |
| `.` | 소수점 |

---

## 구현 조건

- 버튼 클릭 이벤트는 **이벤트 위임**으로 처리합니다. (버튼 각각이 아닌 컨테이너에 등록)
- `event.target.dataset.value`로 버튼 값을 읽습니다.
- 키보드 이벤트는 `document.addEventListener('keydown', ...)`으로 등록합니다.
- `0으로 나누기` 시 "오류" 메시지를 화면에 표시합니다.
- 연속 연산을 지원합니다. (예: 5 + 3 × 2 = 16)

---

## 힌트

```javascript
// 버튼 HTML 예시
<button data-value="7">7</button>
<button data-value="+">+</button>
<button data-value="=">=</button>

// 이벤트 위임
document.getElementById('btnContainer').addEventListener('click', (e) => {
  const btn = e.target.closest('button');
  if (!btn) return;
  const value = btn.dataset.value;
  handleInput(value);
});

// 키보드 이벤트
document.addEventListener('keydown', (e) => {
  const keyMap = {
    '*': '×', '/': '÷', 'Enter': '=', 'Escape': 'C'
  };
  const value = keyMap[e.key] || e.key;
  handleInput(value);
});
```
