# 강의01 과제 — 재사용 가능한 Modal 컴포넌트

## 과제 목표

`isOpen`, `onClose`, `children` props를 받아 동작하는 **재사용 가능한 Modal 컴포넌트**를 만드세요.

## 요구사항

### 필수 props

| prop | 타입 | 설명 |
|------|------|------|
| `isOpen` | `boolean` | `true`이면 모달 표시, `false`이면 숨김 |
| `onClose` | `function` | 모달을 닫는 함수 (버튼 클릭 또는 배경 클릭 시 호출) |
| `children` | `ReactNode` | 모달 내부에 표시할 내용 |

### 선택 props (구현하면 보너스)

| prop | 타입 | 설명 |
|------|------|------|
| `title` | `string` | 모달 제목 |
| `size` | `'sm' \| 'md' \| 'lg'` | 모달 크기 |

### 동작 조건

1. `isOpen`이 `false`이면 아무것도 렌더링하지 않는다.
2. 모달 배경(오버레이)을 클릭하면 `onClose`가 호출된다.
3. 모달 내부를 클릭해도 닫히지 않는다. (이벤트 버블링 방지)
4. 닫기(×) 버튼을 클릭하면 `onClose`가 호출된다.

## 기대 동작

```jsx
// 이렇게 사용할 수 있어야 합니다
function App() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <button onClick={() => setIsOpen(true)}>모달 열기</button>

      <Modal
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        title="확인"
      >
        <p>정말로 삭제하시겠습니까?</p>
        <button onClick={() => setIsOpen(false)}>취소</button>
        <button onClick={() => { alert('삭제 완료!'); setIsOpen(false); }}>삭제</button>
      </Modal>
    </>
  );
}
```

## 힌트

- 오버레이는 `position: fixed`, `inset: 0`으로 화면 전체를 덮습니다.
- 이벤트 버블링 방지는 `e.stopPropagation()` 을 사용하세요.
- `isOpen`이 `false`일 때 `return null`을 사용하면 됩니다.
