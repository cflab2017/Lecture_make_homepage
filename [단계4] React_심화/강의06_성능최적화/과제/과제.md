# 강의06 과제 — 최적화된 검색 필터

## 과제 목표

`useMemo`로 필터링 연산을 최적화하고, `useCallback`으로 이벤트 핸들러를 최적화하는 컴포넌트를 구현하세요.

## 요구사항

### 데이터

```js
// 더미 상품 데이터 (직접 생성)
const PRODUCTS = Array.from({ length: 300 }, (_, i) => ({
  id: i + 1,
  name: `상품 ${i + 1}`,
  category: ['전자제품', '의류', '식품', '도서'][i % 4],
  price: Math.floor(Math.random() * 100000) + 1000,
  inStock: Math.random() > 0.3,
}));
```

### 필터 조건

| 필터 | 타입 | 설명 |
|------|------|------|
| 검색어 | text input | 상품명에 포함 여부 |
| 카테고리 | 버튼 선택 | '전체' 또는 특정 카테고리 |
| 재고 있음만 | checkbox | `inStock === true`인 항목만 |

### 최적화 요구사항

1. **useMemo**: 세 가지 필터 조건(검색어, 카테고리, 재고) 중 하나라도 바뀔 때만 필터링 실행
2. **useCallback**: `handleSearch`, `handleCategoryChange`, `handleStockFilter` 핸들러를 메모이제이션
3. **React.memo**: `ProductCard` 컴포넌트를 `React.memo`로 감싸기

### 표시 정보

- 필터링된 상품 수 (예: "120개 상품")
- 총 가격 합계 (`useMemo`로 계산)
- 각 상품 카드: 이름, 카테고리, 가격, 재고 여부

## 성능 확인 방법
