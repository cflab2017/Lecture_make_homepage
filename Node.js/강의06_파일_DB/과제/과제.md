# 과제 06 — JSON 파일 기반 블로그 API

## 목표

JSON 파일을 데이터베이스로 사용하는 블로그 포스트 CRUD API를 만드세요.
서버를 재시작해도 데이터가 유지되어야 합니다.

---

## 요구사항

### 데이터 구조

```json
{
  "id": 1693000000000,
  "title": "첫 번째 블로그 포스트",
  "content": "포스트 내용입니다.",
  "author": "홍길동",
  "tags": ["node.js", "express"],
  "createdAt": "2025-01-15T14:30:00.000Z",
  "updatedAt": null
}
```

### API 명세

| 메서드 | URL | 설명 |
|--------|-----|------|
| GET | `/api/posts` | 전체 포스트 조회 |
| GET | `/api/posts/:id` | 특정 포스트 조회 |
| POST | `/api/posts` | 포스트 생성 (createdAt 자동 추가) |
| PUT | `/api/posts/:id` | 포스트 수정 (updatedAt 자동 갱신) |
| DELETE | `/api/posts/:id` | 포스트 삭제 |

### 환경 변수 (.env)

```
PORT=3000
NODE_ENV=development
```

### 필수 구현 사항

1. `dotenv`로 환경 변수 로드
2. `data/posts.json` 파일에 데이터 영속화
3. 서버 시작 시 data 폴더 자동 생성
4. `createdAt` 자동 추가 (생성 시)
5. `updatedAt` 자동 갱신 (수정 시)
