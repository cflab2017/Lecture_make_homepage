# 과제 02 — 파일 이름 목록 저장기

## 목표

현재 폴더(또는 지정한 폴더)에 있는 모든 파일 목록을 읽어서 `filelist.txt` 파일에 저장하는 스크립트를 만드세요.

---

## 요구사항

### 기본 기능

1. `fs.readdirSync()`로 현재 디렉토리의 파일/폴더 목록을 읽습니다.
2. 파일과 폴더를 구분하여 표시합니다. (`[파일]` / `[폴더]` 표시)
3. 결과를 `filelist.txt`에 저장합니다.

### 출력 형식 (`filelist.txt` 내용)

```
=== 파일 목록 ===
생성 일시: 2025-01-15 14:30:00

[폴더] node_modules
[파일] package.json
[파일] server.js
[파일] README.md

총 파일 수: 3개
총 폴더 수: 1개
```

### 실행 방법

```bash
node fs-demo.js        # 현재 디렉토리 스캔
node fs-demo.js ./src  # src 폴더 스캔 (도전 과제)
```

---

## 힌트

```js
const fs = require('fs');
const path = require('path');

// 파일/폴더 구분
const stat = fs.statSync(filePath);
stat.isDirectory()  // true면 폴더, false면 파일
stat.isFile()       // true면 파일
```

---

## 도전 과제

- `process.argv[2]`로 스캔할 폴더 경로를 받아 처리
- 파일 크기(bytes)도 함께 출력
- 확장자별로 그룹화하여 출력
