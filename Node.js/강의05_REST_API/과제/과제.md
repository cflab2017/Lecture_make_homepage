# 과제 05 — 메모 REST API

## 목표

메모를 생성, 조회, 수정, 삭제할 수 있는 REST API 서버를 만드세요.

---

## 요구사항

### 데이터 구조

```json
{
  "id": 1693000000000,
  "title": "메모 제목",
  "content": "메모 내용입니다.",
  "createdAt": "2025-01-15T14:30:00.000Z"
}
```

- `id`는 `Date.now()`를 사용합니다.
- `createdAt`은 생성 시점의 ISO 문자열을 사용합니다.

### API 명세

| 메서드 | URL | 설명 | 응답 코드 |
|--------|-----|------|-----------|
| GET | `/api/memos` | 전체 메모 조회 | 200 |
| GET | `/api/memos/:id` | 특정 메모 조회 | 200 / 404 |
| POST | `/api/memos` | 메모 생성 | 201 |
| PUT | `/api/memos/:id` | 메모 수정 | 200 / 404 |
| DELETE | `/api/memos/:id` | 메모 삭제 | 200 / 404 |

### 유효성 검사

- POST, PUT 요청 시 `title`이 없으면 400 Bad Request
- 존재하지 않는 ID 접근 시 404 Not Found

---

## 테스트 방법 (curl)

```bash
# 전체 조회
curl http://localhost:3000/api/memos

# 생성
curl -X POST http://localhost:3000/api/memos \
  -H "Content-Type: application/json" \
  -d '{"title":"오늘 할 일","content":"Node.js 공부하기"}'

# 수정
curl -X PUT http://localhost:3000/api/memos/[id값] \
  -H "Content-Type: application/json" \
  -d '{"title":"수정된 제목","content":"수정된 내용"}'

# 삭제
curl -X DELETE http://localhost:3000/api/memos/[id값]
```

---

## 도전 과제

- `GET /api/memos?title=검색어` — 제목에 검색어가 포함된 메모만 반환
- `GET /api/memos?sort=newest` — 최신 순으로 정렬 (기본값)
- `GET /api/memos?sort=oldest` — 오래된 순으로 정렬
