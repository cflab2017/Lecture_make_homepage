# 과제 03 — 날짜 포맷 변환기

## 목표

`dayjs` 라이브러리를 사용해서 다양한 날짜 형식을 출력하는 스크립트를 만드세요.

---

## 요구사항

### 기본 기능

1. `npm init -y`로 프로젝트를 초기화합니다.
2. `npm install dayjs`로 dayjs를 설치합니다.
3. `npm install --save-dev nodemon`으로 nodemon을 설치합니다.
4. `package.json`의 scripts에 `"dev": "nodemon server.js"`를 추가합니다.

### 출력 내용 (server.js)

아래 정보를 모두 출력하는 스크립트를 작성하세요:

```
=== 오늘의 날짜 정보 ===
기본 형식: 2025-01-15
한국 형식: 2025년 01월 15일 (수요일)
미국 형식: January 15, 2025
짧은 형식: 25/01/15
시간 포함: 2025-01-15 14:30:00

=== D-Day 계산기 ===
오늘부터 100일 후: 2025-04-25
설날(2025-01-29)까지 D-14
크리스마스(2025-12-25)까지 D-344

=== 생년월일 입력 결과 ===
입력한 생년월일: 1995-03-20
오늘 기준 나이: 30세
태어난 지: 10896일째
```

---

## package.json 작성 요령

```json
{
  "name": "date-formatter",
  "version": "1.0.0",
  "scripts": {
    "start": "node server.js",
    "dev": "nodemon server.js"
  },
  "dependencies": {
    "dayjs": "^1.11.10"
  },
  "devDependencies": {
    "nodemon": "^3.0.1"
  }
}
```

---

## 도전 과제

- `process.argv`로 생년월일을 입력받아 나이를 계산
- 이번 달의 모든 날짜를 출력
