# 과제 04 — 도서관 시스템 클래스

## 목표

도서관 시스템을 클래스로 설계하고, 대출과 반납 기능을 구현하세요.

---

## 구현할 클래스

### 1. Book (도서)

```ts
class Book {
  readonly isbn: string;        // ISBN (변경 불가)
  title: string;
  author: string;
  private isLoaned: boolean;    // 대출 여부 (외부에서 직접 변경 불가)
  // ...
}
```

필수 메서드:
- `checkOut()`: 대출 처리 (이미 대출 중이면 오류)
- `returnBook()`: 반납 처리
- `getStatus()`: 현재 상태 반환 ('대출 가능' / '대출 중')

### 2. Member (회원)

```ts
class Member {
  readonly memberId: string;    // 회원번호 (변경 불가)
  name: string;
  private loanedBooks: Book[];  // 대출 중인 도서 목록 (최대 3권)
  // ...
}
```

필수 메서드:
- `borrow(book: Book)`: 도서 대출 (최대 3권 제한)
- `returnBook(book: Book)`: 도서 반납
- `getLoanedBooks()`: 현재 대출 목록 반환

### 3. Library (도서관)

```ts
class Library {
  private books: Book[];
  private members: Member[];
  // ...
}
```

필수 메서드:
- `addBook(book: Book)`: 도서 추가
- `registerMember(member: Member)`: 회원 등록
- `loanBook(memberId: string, isbn: string)`: 회원이 도서 대출
- `returnBook(memberId: string, isbn: string)`: 도서 반납
- `searchBook(keyword: string)`: 제목/저자로 도서 검색

---

## 실행 시나리오

```ts
const library = new Library('서울 중앙 도서관');

// 도서 추가
library.addBook(new Book('978-1', 'TypeScript 완전 정복', '김저자'));
library.addBook(new Book('978-2', 'Node.js 입문', '이저자'));

// 회원 등록
library.registerMember(new Member('홍길동'));

// 대출
library.loanBook('회원ID', '978-1');

// 반납
library.returnBook('회원ID', '978-1');
```
