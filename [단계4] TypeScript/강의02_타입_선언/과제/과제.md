# 과제 02 — 타입 안전한 계산기 함수

## 목표

4칙연산 함수에 TypeScript 타입을 완전히 지정하고, 예외 상황도 타입으로 표현하세요.

---

## 요구사항

### 구현할 함수

```ts
// 1. add — 덧셈
function add(a: ?, b: ?): ? { ... }

// 2. subtract — 뺄셈
function subtract(a: ?, b: ?): ? { ... }

// 3. multiply — 곱셈
function multiply(a: ?, b: ?): ? { ... }

// 4. divide — 나눗셈 (0 나누기 고려)
function divide(a: ?, b: ?): ? { ... }

// 5. calculate — 연산자로 계산 (유니온 타입 활용)
function calculate(a: ?, operator: ?, b: ?): ? { ... }
```

### calculate 함수 예시

```ts
calculate(10, '+', 20)  // 30
calculate(10, '-', 3)   // 7
calculate(10, '*', 5)   // 50
calculate(10, '/', 0)   // null (0으로 나누기)
calculate(10, '%', 3)   // 오류 (지원하지 않는 연산자)
```

---

## 타입 힌트

```ts
// 연산자는 4가지로 제한 — 리터럴 유니온 타입
type Operator = '+' | '-' | '*' | '/';

// 결과는 number 또는 null
function divide(a: number, b: number): number | null { ... }
```

---

## 파일 상단 주석

```ts
// 컴파일: tsc types.ts
// 실행:   node types.js
```

---

## 도전 과제

- `power(base: number, exponent: number): number` — 거듭제곱
- `modulo(a: number, b: number): number | null` — 나머지 연산 (0 나누기 고려)
- `roundTo(value: number, decimals?: number): number` — 소수점 반올림
